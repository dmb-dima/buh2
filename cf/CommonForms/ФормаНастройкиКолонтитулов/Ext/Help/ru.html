Настройка колонтитулов стандартных отчетов

Отчеты ® Настройка колонтитулов стандартных отчетов

Форма позволяет настроить состав и порядок отображения колонтитулов стандартных отчетов при выводе отчетов на печать.

Форма содержит шесть блоков текста (три сверху и три снизу), которые определяют состав верхнего и нижнего колонтитула. Левые блоки текста определяют состав левых краев колонтитулов, центральные блоки текста - центральные части колонтитулов, правые блоки - правые края колонтитулов.

В тексте колонтитула может быть указан произвольный текст. Текст можно ввести по шаблону (кнопка Вставить). При выводе на печать шаблон будет заменен следующими данными:

  • Дата - дата формирования отчета.
  • Время - время формирования отчета.
  • Номер страницы - номер текущей страницы.
  • Страниц всего - общее количество страниц, выводимых на печать.
  • Пользователь - пользователь, сформировавший отчет.
  • Название отчета - название отчета.

В центре формы отображается образец колонтитулов.

Верхний колонтитул будет выводиться на печать, если установлен флажок Выводить верхний колонтитул. При установленном флажке можно задать номер страницы, начиная с которой верхний колонтитул будет выводиться.

Нижний колонтитул будет выводиться на печать, если установлен флажок Выводить нижний колонтитул. При установленном флажке можно задать номер страницы, начиная с которой нижний колонтитул будет выводиться.

Обновить образец отчета можно по кнопке Образец.

Посмотреть, как будут выглядеть колонтитулы на разных страницах отчета, можно по кнопке Отображать страницу.

Записать текущую настройку колонтитулов можно по кнопке ОК.
