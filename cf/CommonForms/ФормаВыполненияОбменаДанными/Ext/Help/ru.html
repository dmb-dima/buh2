Форма "Выполнение обмена данными"

С помощью этой формы можно выполнять обмен данными по произвольной настройке обмена. Для выполнения обмена достаточно выбрать настройку и нажать на кнопку "Выполнить".

Имеется возможность добавить новую настройку обмена данными. Для этого достаточно в выпадающем списке настроек обмена выбрать последний пункт "Добавить новую настройку обмена". После создания новой настройки обмена она автоматически будет выбрана для выполнения.

В меню "Действия" формы имеются пункты:

1. ."Список настроек обмена" - позволяет перейти к списку имеющихся настроек обмена данными,

2. "Применить изменения конфигурации" - позволяет применить изменения к метаданным конфигурации, которые были получены в результате обмена данными. Будет создан и выполнен специальный bat-файл, настроить содержимое которого можно в форме "Настройка файла обновления конфигурации". Рекомендуем использовать автоматическое применение изменений конфигурации в том случае, если вам достоверно известно, что внесенные изменения в конфигурацию не могут отрицательным образом сказаться на ее работоспособности. Если во время обмена данными для произвольной настройки обмена вы получили сообщение об ошибке: "Из главного узла распределенной ИБ получены изменения конфигурации." то только в этом случае, после завершения обмена данными можно воспользоваться автоматическим применением изменений метаданных в конфигурации,

3. "Настройка обновления конфигурации" - позволяет редактировать содержимое bat-файла обновления конфигурации.
