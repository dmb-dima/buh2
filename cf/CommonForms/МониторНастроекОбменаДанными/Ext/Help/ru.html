Монитор обмена данными

Монитор обмена предназначен для отображения информации по обмену данными. Монитор отображает информацию:

1. Результат последней загрузки данных

2. Результат последней выгрузки данных

3. Результат отложенного проведения документов

4. Выполняется ли фоновый обмен данными

 

Информация о последних удачно выполненных операциях:

 

1. Дата посленней удачной загрузки данных

2. Дата последней удачной выгрузки данных

3. Дата последнего удачного отложенного проведения документов

 

Есть возможность проанализировать статистику выполнения обменов данными за определенный период времени, посмотреть историю выполнения обменов данными. На закладке "История коллизий" можно посмотреть коллизии, которые возникали при обмене данными. Когда один и тот же объект был одновременно изменен в двух информационных базах. На закладке "Отложенные движения" можно посмотреть информацию о документах, движения по которым планируется выполнить. При этом инициировать выполнение движений по документам можно интерактивно.
