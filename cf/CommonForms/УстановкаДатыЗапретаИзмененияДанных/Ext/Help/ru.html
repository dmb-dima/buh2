Установка даты запрета изменения данных

Сервис ® Управление пользователями и доступом ® Дата запрета изменения данных

Изменение данных информационной базы ранее определенной даты можно запретить. Для этого нужно установить дату запрета.

Контроль изменения данных выполняется для:

  • всех документов при проведении и отмене проведения;
  • записей регистров накопления, при этом контроль выполняется во всех случаях изменения записей регистров, включая программное.

После указания даты запрета изменения данных, документы за указанную или более раннюю дату будут доступны только для просмотра.

Примечание
Ограничение по дате запрета не распространяется на пользователей, которым доступна роль "Полные права".

Возможны несколько вариантов установки даты запрета изменений:

  • Общая дата запрета изменений устанавливается одновременно для всех организаций и ролей пользователей.
  • Дата запрета по организациям позволяет установить разные даты запрета для организаций, без разделения по ролям.
  • Дата запрета по организациям и пользователям позволяет установить разные даты запрета для организаций по пользователям и (или) группам пользователей. Если пользователь входит в несколько групп пользователей, то в качестве ограничивающей даты берется минимальная из дат, установленных для групп пользователя.
