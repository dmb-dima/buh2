Документ "Сведения о взносах и страховом стаже СПВ-1"

Кадры ® Документы персонифицированного учета ПФР

Документ предназначен для подготовки сведений о работниках достигших пенсионного возраста по форме СПВ-1.

Печатные формы

При вводе документа в шапке обязательно нужно указать следующие реквизиты:

  • Ответственный - работник, ответственный за формирование и передачу данных в ПФР. По умолчанию это ответственный, указанный в настройках пользователя.
  • Отчетный период - период, за который передаются сведения.
  • Номер пачки - номер пачки передаваемых сведений.
  • Тип сведений - Исходные, Корректирующие, Отменяющие.
  • Категория застрахованных лиц -  категория застрахованных лиц (наемный работник и т.п.).

В документе указываются работники, данные которых передаются в ПФР. Необходимо указать:

  • страховой номер;
  • суммы начисленных и уплаченных страховых взносов на обязательное пенсионное страхование;
  • сведения о страховом стаже.

Примечание
По каждому сотруднику должна быть заполнена форма ввода данных СЗВ. Редактировать данные о стаже и страховых взносах работника можно непосредственно в документе по кнопке Открыть данные о стаже.

По кнопке Записать файл на диск откроется окно выбора каталога, в который следует записать файл данных, после чего формируется файл для передачи в ПФР в соответствии с выбранным форматом.

По кнопке Показать файл откроется окно текстового редактора, в которое будет выведен файл для передачи в ПФР, сформированный в соответствии с выбранным форматом файла.

При формировании файлов или печатных форм автоматически проводятся проверки анкетных данных работников, и в случае недостаточности либо ошибочности данных выдаются диагностические сообщения.

Проведение документа (кнопка ОК) означает, что данные переданы в отделение ПФР. При этом сформированные данные фиксируются в информационной базе, при необходимости можно просмотреть все переданные в ПФР данные. 

После того, как отделение ПФР примет переданные сведения, рекомендуется "защитить" документ от случайных изменений. Для этого нужно установить флажок Принято в ПФР.

Печатные формы

Для документа Сведения о страховых взносах и страховом стаже СПВ-1 предусмотрены следующие печатные формы:

  • Сведения о начисленных, уплаченных страховых взносах на обязательное пенсионное страхование и страховом стаже застрахованного лица для установления трудовой пенсии (Форма СПВ-1)
