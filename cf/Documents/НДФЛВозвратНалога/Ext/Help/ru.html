Документ "Возврат НДФЛ"

Зарплата ® Учет НДФЛ и налогов (взносов) с ФОТ ® Возврат НДФЛ

Документ предназначен для регистрации возврата излишне уплаченного налога на доходы по заявлению работников. Суммы возвращаемого налога регистрируются в разрезе ставок налога.

В поле Возврат за указывается год (для прошлых налоговых периодов) или месяц налогового периода (для текущего налогового периода).

В документе указываются работники и суммы излишне уплаченного налога на доходы в разрезе ставок налога.

Список работников, по которым зарегистрирована переплата по налогу на доходы, можно заполнить автоматически по кнопке Заполнить - Работниками, имеющими переплату по НДФЛ или по кнопке Подбор.

Рассчитать сумму переплаты по налогу на доходы можно по кнопке Рассчитать.
