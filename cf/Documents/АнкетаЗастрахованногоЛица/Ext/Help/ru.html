Документ "Анкета застрахованного лица (АДВ-1)"

Кадры ® Документы персонифицированного учета ПФР

Документ предназначен для передачи в отделение ПФР данных формы АДВ-1.

Форма АДВ-1 передается в ПФР по работникам организации, не имеющим страховых номеров ПФР.

Печатные формы

Документ заполняется на основании данных справочника Физические лица.

При вводе документа в шапке обязательно нужно указать следующие реквизиты:

  • Ответственный - работник, ответственный за формирование и передачу данных в ПФР. По умолчанию это ответственный, указанный в настройках пользователя.
  • Номер пачки - номер пачки передаваемых сведений.

В документе указываются работники, данные которых передаются в ПФР для присвоения страховых номеров.

Список работников организации, не имеющих страховых номеров ПФР, и данные для формы АДВ-1 можно заполнить по кнопке Заполнить - Работниками без страховых номеров.

По кнопке Записать файл на диск откроется окно выбора каталога, в который следует записать файл данных, после чего формируется файл для передачи в ПФР в соответствии с выбранным форматом.

По кнопке Показать файл откроется окно текстового редактора, в которое будет выведен файл для передачи в ПФР, сформированный в соответствии с выбранным форматом файла.

При формировании файлов или печатных форм автоматически проводятся проверки анкетных данных работников, и в случае недостаточности либо ошибочности данных выдаются диагностические сообщения.

Проведение документа (кнопка ОК) означает, что данные переданы в отделение ПФР. При этом сформированные данные фиксируются в информационной базе и можно просмотреть все переданные в ПФР данные.

Печатные формы
