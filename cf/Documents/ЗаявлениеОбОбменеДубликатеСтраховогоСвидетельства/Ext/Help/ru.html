Документ "Заявление об обмене, дубликате страхового свидетельства"

Кадры ® Документы персонифицированного учета ПФР

Документ предназначен для передачи в отделение ПФР форм АДВ-2 и АДВ-3.

Форма АДВ-2 передается в ПФР в случае изменения анкетных данных работников, например, если сотрудница после замужества поменяла фамилию и документ, удостоверяющий личность.

Форма АДВ-3 передается в ПФР в случае потери работником страхового свидетельства.

Печатные формы

При вводе документа в шапке обязательно нужно указать следующие реквизиты:

  • Ответственный - работник, ответственный за формирование и передачу данных в ПФР. По умолчанию это ответственный, указанный в настройках пользователя.
  • Вид заявлений - вид заявления для передачи в ПФР (по форме АДВ-2 или АДВ-3).
  • Номер пачки - номер пачки передаваемых сведений.

После выбора вида заявления нужно заполнить закладки:

  • Работники
  • Данные, указанные в свидетельстве
  • Изменившиеся данные

На закладке Работники указывается список работников, по которым передаются формы.

  • В колонке Дата получения свидетельства указывается дата последнего по времени получения свидетельства этим работником, если это произошло в нашей организации, иначе поле не заполняется.

Закладки Данные, указанные в свидетельстве и Изменившиеся данные заполняются автоматически по кнопке Заполнить - Заполнить заявления. При этом данные на закладке Изменившиеся данные заполняются только для тех работников, для которых была указана дата получения свидетельства и в информационной базе присутствует информация на этих работников, переданная ранее документами Анкета застрахованного лица или Заявление об обмене, дубликате страхового свидетельства.

По кнопке Записать файл на диск откроется окно выбора каталога, в который следует записать файл данных, после чего формируется файл для передачи в ПФР в соответствии с выбранным форматом.

По кнопке Показать файл откроется окно текстового редактора, в которое будет выведен файл для передачи в ПФР, сформированный в соответствии с выбранным форматом файла.

При формировании файлов или печатных форм автоматически проводятся проверки анкетных данных работников, и в случае недостаточности либо ошибочности данных выдаются диагностические сообщения.

Проведение документа (кнопка ОК) означает, что данные переданы в отделение ПФР. При этом сформированные данные фиксируются в информационной базе и можно просмотреть все переданные в ПФР данные.

Печатные формы

Для документа Заявление об обмене, дубликате страхового свидетельства предусмотрены следующие печатные формы:

  • Заявление об обмене страхового свидетельства (Форма АДВ-2)
  • Заявление о выдаче дубликата страхового свидетельства (Форма АДВ-3)

Выбор печатной формы осуществляется автоматически в соответствии со значением реквизита Вид заявлений.
