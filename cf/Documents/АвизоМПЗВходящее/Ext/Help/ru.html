Документ "Авизо по МПЗ входящее"

Склад ® Авизо по МПЗ входящее

Документ предназначен для отражения поступления товарно-материальных ценностей от обособленного подразделения, выделенного на отдельный баланс.

Документ может быть введен на основании документа Авизо по МПЗ исходящее.

При вводе документа в шапке нужно указать следующие реквизиты:
  • Организация - обособленное подразделение, на котором будут числиться поступившие МПЗ.
  • Отправитель - обособленное подразделение, с баланса которого переданы МПЗ.
  • Счет расчетов - счет внутрихозяйственных расчетов, субсчет счета 79 "Внутрихозяйственные расчеты".
  • Основание - документ Авизо по МПЗ исходящее, которым переданы МПЗ.
  • Склад и Подразделение - склад и производственное подразделение, на котором будут числиться поступившие МПЗ.

В табличной части нужно заполнить список поступивших товарно-материальных ценностей .

  • Заполнить список можно по кнопке Подбор.
  • Сведения о поступивших МПЗ можно заполнить автоматически по документу-основанию по кнопке Заполнить - По документу основанию.
