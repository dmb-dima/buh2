Документ "Регистрация оплаты НДС в бюджет"

Покупка ® Ведение книги покупок ® Регистрация оплаты НДС в бюджет

Документ предназначен для регистрации оплаты в бюджет сумм НДС по договорам налогового агента и при выполнении строительно-монтажных работ для собственного потребления (хозяйственным способом). Регистрация оплат требуется для отражения НДС к вычету по данным операциям.

По договорам налогового агента

Закладка По договорам налогового агента используется только при расчетах по договорам, в которых указано, что НДС при исполнении обязанностей налогового агента начисляется при поступлении ценностей. В ином случае документ для регистрации оплаты НДС по договорам налогового агента использовать не следует.

Закладка заполняется еще не оплаченными счетами-фактурами, по которым был начислен НДС к уплате в бюджет по договорам с исполнением обязанностей налогового агента. Закладка заполняется автоматически по кнопке Заполнить.

В реквизите Документ оплаты указывается платежный документ, которым оплачен НДС в бюджет.

Для собственного потребления

Закладка Для собственного потребления заполняется еще не оплаченными счетами-фактурами, по которым был начислен к уплате в бюджет НДС при выполнении строительно-монтажных работ для собственного потребления. Закладка заполняется автоматически по кнопке Заполнить.

В реквизите Документ оплаты указывается платежный документ, которым оплачен НДС в бюджет.
