Документ "Прием на работу"

Кадры ® Прием на работу

Документ предназначен для оформления приказа о приеме на работу.

Помощник приема на работу предназначен для быстрого ввода сведений о новом сотруднике. Помощник приема на работу можно отключить в настройках пользователя (флажок Не использовать помощника приема на работу).

 Советы

Как ввести приказ о приеме сотрудника на работу

Как ввести информацию о вычетах по сотруднику для расчета НДФЛ

В документе предусмотрено два режима работы:

  • прием на работу одного сотрудника (основной режим, вводятся сведения только на одного сотрудника);
  • прием на работу группы сотрудников (включается установкой флажка Приказ на группу сотрудников, становятся доступными закладки Работники и Начисления).

В документе нужно указать сотрудника, его табельный номер, дату приема, подразделение и сведения для расчета зарплаты (вид начисления и его размер).

Примечание
При необходимости назначить несколько видов начислений, нужно установить флажок Приказ на группу сотрудников и ввести список начислений на закладке Начисления.

Для документа Прием на работу предусмотрены следующие печатные формы:

  • Форма Т-1 Приказ (распоряжение) о приеме работника на работу
  • Форма Т-1а Приказ (распоряжение) о приеме работников на работу
