Документ "Авизо по расчетам входящее"

Покупка (Продажа) ® Авизо по расчетам входящее

Документ предназначен для отражения поступления дебиторской и кредиторской задолженности от обособленного подразделения, выделенного на отдельный баланс.

Документ может быть введен на основании документа Авизо по расчетам исходящее.

Одним документом может быть принято несколько сумм дебиторской и кредиторской задолженности по разным контрагентам и договорам.

При вводе документа в шапке нужно указать следующие реквизиты:

  • Организация - обособленное подразделение, на котором будет числиться принятая задолженность.
  • Отправитель - обособленное подразделение, с баланса которого была передана задолженность.
  • Счет расчетов - счет внутрихозяйственных расчетов, субсчет счета 79 "Внутрихозяйственные расчеты".
  • Основание - документ Авизо по расчетам исходящее, которым была передана задолженность подразделением-отправителем.
  • Флажок Авизо по расчетам в валюте необходимо установить при поступлении задолженности по валютным договорам.

В табличной части нужно заполнить реквизиты:

  • Вид задолженности - выбрать значение из списка: Дебиторская или Кредиторская.
  • Контрагент - контрагент, задолженность по которому принимается.
  • Договор - договор, задолженность по которому принимается.
  • Счет учета - счет учета, на котором будет числиться принятая задолженность.
  • Сумма - сумма принятой задолженности.

Суммы принятой задолженности можно заполнить автоматически по документу-основанию по кнопке Заполнить - По документу основанию.
