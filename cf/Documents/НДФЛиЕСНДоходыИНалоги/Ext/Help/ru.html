Документ "Ввод доходов, НДФЛ и налогов (взносов) с ФОТ"

Зарплата ® Данные учета зарплаты во внешней программе ® Ввод доходов, НДФЛ и налогов (взносов) с ФОТ

Документ предназначен для учета данных по НДФЛ и налогам (взносам) с ФОТ, если учет заработной платы ведется во внешней программе. Документ также предназначен для ввода вручную данных по НДФЛ по ставке, отличной от ставки 13 %.

Данные по НДФЛ отражаются в Налоговых карточках 1-НДФЛ и используются при формировании справок по форме 2-НДФЛ для представления в ИФНС и выдачи на руки работникам.

Данные по страховым взносам можно проанализировать с помощью Индивидуальных карточек по страховым взносам.

Данные о суммах пособий по социальному страхованию используются при формировании Расчетных ведомостей по средствам ФСС по формам 4-ФСС и 4а-ФСС.

Данные по ЕСН отражаются в Индивидуальных карточках и Сводных карточках по ЕСН.
