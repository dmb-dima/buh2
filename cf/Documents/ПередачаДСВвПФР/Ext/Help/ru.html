Документ "Передача документов ДСВ-1 в ПФР"

Кадры ® Документы персонифицированного учета ПФР

Документ предназначен для передачи в ПФР заявлений о добровольном вступлении в правоотношения (форма ДСВ-1).

Форма ДСВ-1 передается в ПФР по работникам организации, желающим уплачивать добровольные страховые взносы в ПФР.

При вводе документа в шапке обязательно нужно указать следующие реквизиты:

  • Ответственный - работник, ответственный за формирование и передачу данных в ПФР. По умолчанию это ответственный, указанный в настройках пользователя.
  • Наименование ПФР - наименование территориального органа ПФР, в который передаются сведения.
  • Номер пачки - номер пачки передаваемых сведений.

В документе указываются работники, заявления которых передаются в ПФР для регистрации.

Список работников можно заполнить по кнопке Подбор или ввести вручную. В этом случае автоматически заполняются все сведения кроме даты заполнения заявления.

По кнопке Записать файл на диск откроется окно выбора каталога, в который следует записать файл данных, после чего формируется файл для передачи в ПФР.

По кнопке Показать файл откроется окно текстового редактора, в которое будет выведен файл для передачи в ПФР.

При формировании файлов или печатных форм автоматически проводятся проверки анкетных данных работников, и в случае недостаточности либо ошибочности данных выдаются диагностические сообщения.

Проведение документа (кнопка ОК) означает, что данные переданы в отделение ПФР. При этом сформированные данные фиксируются в информационной базе, и имеется возможность просмотреть все переданные в ПФР данные.
