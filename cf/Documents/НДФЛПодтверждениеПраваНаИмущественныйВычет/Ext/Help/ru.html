Документ "Подтверждение права на имущественный вычет"

Зарплата ® Учет НДФЛ и налогов (взносов) с ФОТ ® Подтверждение права на имущественный вычет

Документ предназначен для регистрации суммы имущественного вычета, в соответствии с представленным работником уведомлением о праве на получение имущественного налогового вычета по налогу на доходы физических лиц.

В поле Возврат за указывается год, за который предоставляется налоговый вычет.

  • Работник - сотрудник организации.
  • Дата - дата регистрации уведомления в организации.
  • Расходы, %% по кредитам, Код ИФНС, Номер уведомления и Дата уведомления - заполняются соответствующими данными из уведомления ИФНС о праве на получение имущественного налогового вычета по налогу на доходы физических лиц.
