Документ "Авизо по прочим операциям исходящее"

Операция ® Прочие операции ® Авизо по прочим операциям исходящее

Документ предназначен для отражения передачи данных, для которых не предусмотрено других документов авизо. Передача осуществляется между обособленными подразделениями, выделенными на отдельный баланс.

Авизо по ОС исходящее
Авизо по МПЗ исходящее

Одним документом может быть передано несколько сумм по разным счетам бухгалтерского учета.

При вводе документа в шапке нужно указать следующие реквизиты:

  • Организация - обособленное подразделение, которое передает учетные данные.
  • Получатель - обособленное подразделение, на баланс которого передаются учетные данные.
  • Счет расчетов - счет внутрихозяйственных расчетов, субсчет счета 79 "Внутрихозяйственные расчеты".

По передаваемым на баланс суммам нужно указать Счет учета и аналитику по этому счету.
