Документ "Авизо по ОС входящее"

ОС ® Авизо по ОС входящее

Документ предназначен для отражения поступления основных средств от обособленного подразделения, выделенного на отдельный баланс.

Документ может быть введен на основании документа Авизо по ОС исходящее.

При вводе документа в шапке нужно указать следующие реквизиты:

  • Организация - обособленное подразделение, на котором будет числиться поступившее основное средство.
  • Отправитель - обособленное подразделение, с баланса которого передано основаное средство.
  • Счет расчетов - счет внутрихозяйственных расчетов, субсчет счета 79 "Внутрихозяйственные расчеты".
  • Основание - документ Авизо по ОС исходящее, которым передано основное средство.
  • Событие - событие основного средства, которое отражается документом.

В табличной части необходимо заполнить список поступивших основных средств и сведения о них.

  • Заполнить список можно по кнопке Подбор.
  • Сведения об основном средстве можно заполнить автоматически по документу-основанию по кнопке Заполнить - По документу основанию.
