Документ "Кадровое перемещение"

Кадры ® Кадровое перемещение

Документ предназначен для оформления приказа о переводе работника организации на другую работу или в другое подразделение, а также об изменении оплаты труда.

В документе предусмотрено два режима работы:

  • перемещение одного сотрудника (основной режим, вводятся сведения только на одного сотрудника);

  • перемещение группы сотрудников (включается установкой флажка Приказ на группу сотрудников, становятся доступными закладки Работники и Начисления).

В документе нужно указать сотрудника, его табельный номер, дату перевода в новое подразделение или на новую должность, а также список изменяемых видов начислений (вид расчета и его размер).

Список сотрудников можно заполнить автоматически по кнопке Заполнить ® Списком работников или по кнопке Подбор - табличная часть заполнится плановыми начислениями работника на дату документа.

В табличной части Изменение сведений для расчета зарплаты  (закладка Начисления - для приказа на группу сотрудников) нужно указать:

  • Начисление - вид начисления, по которому регистрируется изменение.
  • Действие - действие, которое необходимо произвести для указанного начисления:
    • Начать - назначение сотруднику нового начисления,
    • Изменить - изменение размера указанного начисления,
    • Прекратить - прекратить действие указанного начисления,
    • Не изменять - изменения для указанного начисления не регистрируются.
  • Размер - размер начисления.

Примечание
Если нужно не просто изменить размер текущего начисления, а заменить одно начисление на другое, то в табличной части Изменение сведений для расчета зарплаты  (закладка Начисления - для приказа на группу сотрудников) нужно указать новое начисление и прекратить предыдущее.

На одну и ту же дату может действовать только одно начисление данного вида. Например, на первое число текущего месяца на одного работника можно ввести только одно начисление "Премия". Можно задать вторую премию другим видом начисления, например, "Премия за перевыполнение плана". Оба вида начислений могут действовать с одной даты.

Для  документа Кадровое перемещение предусмотрены следующие печатные формы:

  • Форма Т-5 Приказ (распоряжение) о переводе работника на другую работу
  • Форма Т-5а Приказ (распоряжение) о переводе работников на другую работу
