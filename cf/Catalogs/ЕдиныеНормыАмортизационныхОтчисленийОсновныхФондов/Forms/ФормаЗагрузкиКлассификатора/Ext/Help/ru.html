Форма "Загрузка классификатора ЕНАОФ"

С помощью этой формы можно загрузить классификатор ЕНАОФ из XML файла данных.  Файл данных "enaof.xml" входит в комплект поставки конфигурации.

Возможна полная загрузка или загрузка выбранных групп.

В диалоговом окне нужно выбрать XML файл данных. После нажатия кнопки "Прочитать файл" файл данных будет прочитан и группы классификатора будут отображены на форме в виде дерева.

С помощью кнопки "Установить пометки" можно установить пометки на все группы.

С помощью кнопки "Снять  пометки" можно снять пометки со всех групп.

В процессе загрузки осуществляется поиск по коду в справочнике. Если  установлен флажок "Переписывать объекты" -  найденный в справочнике элемент (группа) будет переписан соответствующим элементом (группой) из классификатора. По умолчанию флажок сброшен.

Загрузка классификатора в справочник производится при нажатии на кнопку "Загрузить". При этом будут загружены помеченные группы, их элементы  и все родители помеченных групп.

Если в справочнике существуют элементы с теми же кодами, как и у групп классификатора - загрузка соответствующих групп и подгрупп не производится.

Если в справочнике существуют группы с теми же кодами, как и у элементов классификатора - загрузка соответствующих элементов не производится.
