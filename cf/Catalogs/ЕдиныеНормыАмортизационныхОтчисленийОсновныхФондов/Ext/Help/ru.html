Справочник "ЕНАОФ"

ОС ® Классификатор ЕНАОФ

Справочник содержит классификатор основных средств, по которым установлены стандартные шифры и годовые нормы амортизационных отчислений.

По этому справочнику классифицируются основные средства, для которых рассчитывается износ по ЕНАОФ.

Для автотранспорта используются нормы амортизационных отчислений в процентах от стоимости автомобиля в расчете на 1000 км пробега.

Шифр по ЕНАОФ указывается для основные средства в поле Шифр по ЕНАОФ.
