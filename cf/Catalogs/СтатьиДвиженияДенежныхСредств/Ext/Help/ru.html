Справочник "Статьи движения денежных средств"

Банк (Касса) ® Статьи движения денежных средств

Справочник предназначен для хранения списка статей движения денежных средств по видам и группам.

Справочник используется для ведения аналитического учета на счетах учета денежных средств для автоматического заполнения формы №4 "Отчет о движении денежных средств".

Примечание
Признак ведения учета по статьям движения денежных средств устанавливается в настройках параметров учета. Если организации не нужно формировать форму №4 "Отчет о движении денежных средств", то можно не вести учет денежных средств по статьям движения.

При вводе статей движения нужно выбрать вид движения. Вид движения соответствует показателям формы №4 "Отчет о движении денежных средств". Этот реквизит можно использовать для анализа данных в стандартных отчетах.

Справочник имеет многоуровневую, иерархическую структуру. Элементы справочника можно перегруппировать, перенося их из группы в группу.
