Справочник "Конвертации из информационных баз предыдущих версий"

Справочник предназначен для хранения сведений о конфигурациях на платформе "1С:Предприятие 7.7", а также о версиях конфигурации "Бухгалтерия предприятия" (ред. 1.6), для которых поддерживается автоматический перенос данных при помощи обработок Перенос данных из информационных баз 1С:Предприятия 7.7, Перенос данных из информационных баз 1С:Бухгалтерии 8.

В справочнике выводится информация о наименовании конфигурации, ее внутреннем идентификаторе, номере релиза конфигурации (перенос поддерживается только для релизов конфигураций не ниже указанного в справочнике), а также имена файлов выгрузки данных и правил конвертации.

По кнопке Записать файлы конвертации на диск производится сохранение файлов обработки конвертации данных и файла правил для выбранной конфигурации. Файлы сохраняются с теми именами, которые указаны в справочнике.

Если предполагается производить выгрузку данных из конфигураций на платформе "1С:Предприятие 7.7" при помощи Помощника перехода на 1С:Бухгалтерию 8, рекомендуется предварительно сохранять файлы конвертации из справочника в каталог ExtForms каталога информационной базы, из которой предполагается выгружать данные.

По кнопке Заменить файлы конвертации производится загрузка внешних файлов обработки конвертации данных и файла правил для выбранной конфигурации. Для загрузки следует указать каталог, в котором содержатся внешние файлы обработки конвертации данных и правил . Если в информационную базу загружены внешние файлы обработки конвертации данных и правил, то при проведении выгрузки данных обработками, перехода типовые правила конвертации использоваться не будут.

По кнопке Восстановить фалы конвертации загруженные файлы обработки конвертации данных и правил удаляются. При выгрузке данных обработками перехода будут использоваться типовые правила конвертации.
