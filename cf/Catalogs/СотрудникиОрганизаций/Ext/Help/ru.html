Справочник "Сотрудники организаций"

Кадры ® Сотрудники организаций

Справочник предназначен для хранения списка сотрудников организаций. Одно физическое лицо может являться одновременно сотрудником нескольких организаций. Перейти к данным физического лица можно по гиперссылке Более подробно о физическом лице.

Ввод нового сотрудника

Оформить приказ о приеме на работу (форма Т-1) можно по кнопке Добавить.

Помощник приема на работу предназначен для быстрого ввода сведений о новом сотруднике. Помощник приема на работу можно отключить в настройках пользователя (флажок Не использовать помощника приема на работу).

Настройка списка сотрудников

Список сотрудников организаций можно:

  • отсортировать по одной организации или отобразить по всем организациям (поле Организация);
  • упорядочить по наименованию, подразделению, должности, дате приема на работу или табельному номера (кнопка Упорядочить по);
  • отобразить иерархически по подразделениям организации или по группам. Включить иерархию справочника можно по кнопке Использовать иерархию командной панели. В открывшемся слева окне можно задать параметры иерархии.

Дополнительные возможности

Приказ о переводе сотрудника на другую работу (форма Т-5) или приказ о прекращении трудового соглашения с работником (форма Т-8) можно оформить по кнопке Создать документ.

Данные кадрового учета и заработной платы сотрудника можно проанализировать в обработке Управление данными работника (кнопка Данные сотрудника - Управление данными работника).

Личную карточку (форма Т-2), Заявление ДСВ-1 можно сформировать по кнопке Данные сотрудника - Личная карточка Т-2 (Заявление ДСВ-1).

Для передачи данных СЗВ в Пенсионный фонд необходимо по каждому сотруднику заполнить сведения о стаже и страховых взносах работника в форме ввода данных СЗВ. Заполнить сведения можно по кнопке Данные сотрудника - Ввод данных СЗВ4.
