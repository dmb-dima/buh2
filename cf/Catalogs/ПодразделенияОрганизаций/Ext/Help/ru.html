Справочник "Подразделения организаций"

Предприятие ® Подразделения организаций

Справочник предназначен для хранения списка подразделений организации, в разрезе которых ведется аналитический учет в программе.

Справочник позволяет вести учет затрат на местах производства работ (цех №1, цех №2) и по местам концентрации хозяйственных функций (дирекция, бухгалтерия).

Обособленные подразделения

Справочник также используется для регистрации обособленных подразделений, не выделенных на отдельный баланс. Обособленные подразделения, выделенные на отдельный баланс, регистрируются в справочнике Организации.

Для регистрации сведений об обособленном подразделении нужно установить флажок Обособленное подразделение и заполнить сведения на закладках:

  • Основные сведения

  • Ответственные лица

  • Адреса и телефоны

На закладке Основные сведения указываются:

  • наименование подразделения для заполнения в первичных документах,
  • префикс подразделения,
  • районный коэффициент,
  • сведения о регистрации в ИФНС - необходимо выбрать значение в поле Регистрация, остальные реквизиты заполнятся автоматически.

На закладке Ответственные лица нужно заполнить ответственных лиц обособленного подразделения, данные которых будут заполняться по умолчанию в печатных формах.

На закладке Адреса и телефоны заполняется контактная информация обособленного подразделения, которая будет заполняться по умолчанию в печатных формах.
