Справочник "Виды взаиморасчетов"

Справочник предназначен для классификации однотипных договоров контрагентов для дополнительного разделения расчетов с контрагентами.

Например, для долгосрочных договоров можно установить вид взаиморасчетов - долгосрочные. Организация, предоставляющая услуги связи, может вести расчеты с контрагентами раздельно по видам взаиморасчетов "Телефонная связь", "Интернет", "Беспроводная связь".

Вид взаиморасчетов можно использовать для анализа данных в стандартных отчетах.
