Справочник "Группы пользователей"

Сервис ® Управление пользователями и доступом ® Группы пользователей

Справочник предназначен для хранения групп пользователей. Группы пользователей информационной базы можно использовать для ограничения доступа, например, при установке даты запрета изменения данных.

При создании группы нужно заполнить состав группы.

Настроить права доступа для группы пользователей можно по кнопке Перейти справочника или элемента справочника.

Просмотреть группы, в которые входит пользователь, можно в настройках пользователя (кнопка Группы пользователя).
