Статьи затрат по налогам (взносам) с ФОТ

Зарплата ® Сведения о начислениях ® Статьи затрат по налогам (взносам) с ФОТ

Регистр предназначен для хранения списка статьей затрат для отражения в учете налогов (взносов) с ФОТ, отличных от статей затрат для отражения расходов по оплате труда. По умолчанию, налоги (взносы) с ФОТ отражаются на счетах затрат по той же статье затрат, что и оплата труда.

Для отражения в учете налогов (взносов) с ФОТ по статьям затрат, отличным от статьи затрат по оплате труда, необходимо настроить соответствие статей затрат.

При вводе строки соответствия указывается статья затрат по оплате труда, подмену которой необходимо произвести. Далее задается вид расхода по налогам с ФОТ и статья затрат, на которую должны быть отнесены налоги (взносы) с ФОТ. Если не указать статью затрат по оплате труда, то все налоги и отчисления с ФОТ, относящиеся к заданному виду расхода по налогам (взносам) с ФОТ будут отражаться по статье затрат, указанной в колонке Статья затрат по налогам с ФОТ.

С 2010 г. в регистре нужно установить соответствие статей затрат по начислению отплаты труда и налогов (взносов) с ФОТ по видам расхода:

  • Страховые взносы
  • ФСС НС

Вид расхода Страховые взносы используется для соответствия статей затрат по страховым взносам.

Вид расхода ФСС НС используется для соответствия статей затрат по отчислениям в ФСС на страхование от несчастных случаев на производстве и профессиональным заболеваниям.

По видам расхода ЕСН, Отчисления в ПФР, ФСС БЛ нужно задать соответствие для статей затрат, используемых для начисления налогов (взносов) до 2010 г.
