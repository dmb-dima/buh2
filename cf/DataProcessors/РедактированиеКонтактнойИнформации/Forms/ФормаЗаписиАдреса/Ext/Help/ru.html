Форма редактирования контактной информации. Адрес.

Форма предназначена для записи и редактирования адресов объектов информационной базы.

В поле "Объект", выбирается объект информационной базы, для которого вводится адрес:

  • Контрагенты

  • Контактные лица

  • Физические лица

  • Организации

  • Пользователи

    При вводе нового адреса из таблицы контактной информации в форме объекта, объект подставляется автоматически.


    Вид адреса может быть 2-х типов:

    • Общий
    • Персональный

    Общий вид контактной информации.

    Общий вид контактной информации выбирается из справочника "Виды контактной информации". В данном справочнике хранятся виды контактной информации, которые присущи большинству объектов контактной информации одного типа (контрагенты, пользователи и пр.).

    Персональный вид контактной информации.

    Персональный вид контактной информации определяется только для текущего объекта. При начале ввода наименования вида контактной информации программа производит подстрочный поиск в справочнике "Виды контактной информации". Если элемент (или элементы), соответствующий введенному наименованию, найден, то система ведет себя стандартно. Если найден единственный элемент вида контактной информации данного типа и выбранного типа объекта - то он автоматически подставляется. Если найдено несколько - то предлагается выбрать из списка. В обоих случаях выбранное значение вида контактной информации является общим. Если программа не нашла ни одного вида контактной информации по введенному наименованию, то цвет текста становится синим, это означает что данный вид контактной информации является персональным для текущего объекта.


    Пользователь может дополнительно определить, какой адрес он вводит:

    • Российский адрес
    • Адрес за пределами РФ

    Российский адрес вводится в формате МНС, при этом значения всех полей можно выбирать из адресного классификатора, используя соответствующие кнопки выбора полей ввода. Адресный классификатор можно загрузить в информационную базу. При нажатии кнопки "Классификатор" открывается форма настройки загрузки. В выпадающих списках каждого поля ввода можно выбрать адресные сокращения (ул., пос., пр-т. и пр.) для выбранного значения части адреса. Представление адреса сформируется автоматически и пользователь изменить его не может.

    Адрес за пределами РФ вводится пользователем в произвольном формате.

    При нажатии кнопки ОК производится запись всех изменений в регистр сведений "Контактная информация".
