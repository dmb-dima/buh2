Зарплата ® Выплата зарплаты ® Выплата зарплаты расходными ордерами

Обработка предназначена для автоматического формирования расходных кассовых ордеров для выдачи зарплаты работникам. Обработку можно ввести на основании документа Ведомость на выплату зарплаты.

Как отразить в учете выплату зарплаты сотрудникам расходными кассовыми ордерами

В поле Платежная ведомость следует указать документ, на основании которого выплачивается зарплата. Табличная часть заполнится автоматически.

Сформировать список расходных кассовых ордеров по отмеченным работникам можно по кнопке Создать документы.

Провести созданные документы Расходный кассовый ордер можно по кнопке Провести документы.

По одной ведомости создавать и проводить расходные кассовые ордера можно в разное время. В этом случае уже созданные по документу расходные кассовые ордера будут автоматически найдены и заполнены в списке.
