Импорт / экспорт операций по лицевым счетам работников

Зарплата ® Выплата зарплаты ® Выплата зарплаты через банк ® Импорт / экспорт операций по лицевым счетам работников

Обработка предназначена для выгрузки в банк документов Заявка на открытие счетов и Ведомость на выплату зарплаты, а также для загрузки пришедших из банка документов Подтверждение открытия счетов и Подтверждение зачисления зарплаты.

На закладке Экспорт заявок на открытие счетов нужно указать каталог экспорта, куда будут выгружены документы. В списке заявок на открытие счетов нужно отметить только те документы, которые необходимо выгрузить. Выгрузка осуществляется по кнопке Выгрузить. В результате будет сформирован файл *.xml с документами Заявка на открытие счетов. Имя файла определяется автоматически.

На закладке Экспорт зачисления зарплаты нужно указать каталог, куда будут выгружены документы. Далее нужно заполнить список ведомостей на перечисление зарплаты, отметить те документы, которые необходимо выгрузить. Выгрузка осуществляется по кнопке Выгрузить. В результате будет сформирован файл *.xml с документами Ведомость на выплату зарплаты. Имя файла определяется автоматически.

На закладке Импорт указывается каталог, где располагаются файлы, пришедшие из банка. Необходимо заполнить "Файлы для импорта" и отметить те файлы, из которых нужно импортировать данные. На основании данных, содержащихся в отмеченных файлах будут проведены оформленные ранее документы Заявки на открытие счетов. При импорте подтверждения зачисления зарплаты будут созданы документы Списание с расчетного счета.

На закладке Протоколы обмена содержится список протоколов обмена с банком.
