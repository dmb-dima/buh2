Расшифровка регламентированной отчетности

Для форм регламентированной отчетности, которые могут заполняться автоматически по данным информационной базы, существует возможность показать алгоритм формирования значений показателей  отчета. Такой режим называется «расшифровкой». Режим доступен не для всех форм отчетности: формальным признаком возможности расшифровки для конкретного регламентированного отчета является присутствие кнопки «Расшифровка» в верхней командной панели формы отчета. Расшифровка доступна после того, как отчет автоматически заполнен по данным информационной базы. Информация для расшифровки накапливается в процессе автозаполнения отчета.

Форма расшифровки содержит таблицу с алгоритмом вычисления значения показателя, для которого вызвана расшифровка, или несколько таблиц - для всех показателей формы отчетности. Каждая строка таблицы содержит описание одного из слагаемых и значение этого слагаемого. Итоговое значение показателя выдается в нижней части таблицы. 

Форма расшифровки может иметь несколько закладок для разных показателей, если расшифровка вызывалась несколько раз.

Из расшифровки можно вызывать стандартные отчеты для более детального раскрытия сумм отчета. Если при помещении курсора мыши на суммы в окне расшифровки курсор принимает вид лупы, тогда двойным щелчком мыши будет вызван один из стандартных отчетов – оборотно-сальдовая ведомость по счету или анализ счета, содержащий данные счета/корсчета, участвовавших в формировании текущего слагаемого значения показателя.

В нижней командной панели формы расшифровки расположен ряд кнопок для управления расшифровкой.

 «Активизировать родительский отчет» - кнопка активизирует отчет, для которого была вызвана эта расшифровка.

«Печать» - по нажатию это кнопки выдается запрос для выбора, расшифровку каких показателей следует печатать. После установки флажков для требуемых показателей и нажатия кнопки «ОК» будет сформированы печатные формы для выбранных показателей.

«Сохранить» - кнопка позволяет сохранить расшифровку в виде файла формата табличного документа «1С:Предприятия 8».

«Закрыть» - закрывает окно расшифровки.
