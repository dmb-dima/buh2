Групповая перепроведение документов

Сервис ® Групповая перепроведение документов 

Обработка предназначена для группового перепроведения документов. Например, для восстановления правильной последовательности проведения документов.

Перепровести документы можно по всем организациям или по выбранным организациям.

При выполнении можно останавливать перепроведение при выявлении ошибки, для этого нужно установить соответствующий флажок.

Перепроведение выполняется по кнопке Выполнить.
