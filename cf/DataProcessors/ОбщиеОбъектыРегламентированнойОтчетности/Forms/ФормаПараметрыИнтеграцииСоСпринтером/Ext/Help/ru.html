Параметры интеграции c ПК «Спринтер» компании «Такском»

Форма настройки параметров интеграции конфигурации с ПК «Спринтер» компании «Такском» содержит следующие параметры:

Логин

Логин – индивидуальный код данного абонента-налогоплательщика в данной налоговой инспекции. Код абонента состоит из 7 символов (цифры и латинские буквы от A до Z): первые 4 символа - код ИФНС, остальные 3 символа - код абонента.
Код абонента присваивается абоненту-налогоплательщику организацией - провайдером телекоммуникационных услуг и сообщается абоненту во время выдачи ему носителя ключевой информации. Логин указан в карточке настройки, выдаваемой в компании «Такском» при подключении к системе.

При изменении ранее введенного логина имя каталога для отправки данных отчетности формируется заново. Поэтому при смене логина следует проверить сформированное автоматически имя каталога для отправки данных и, если необходимо, отредактировать его.

Каталог программы электронной почты

В поле «Каталог программы электронной почты» необходимо указать каталог, в котором расположена программа электронной почты. По умолчанию в этом поле указывается каталог C:\DIPOST .

Если при установке программного обеспечения вы не меняли каталогов, предлагаемых программами установки, можно не изменять указанный каталог программы электронной почты.

Имя каталога программы электронной почты можно не вводить вручную, а, нажав кнопку «...» справа от поля для ввода имени каталога, открыть диалог для выбора нужного каталога. Перед выполнением выгрузки программу электронной почты следует настроить на каталог выгрузки, имя которого указано в поле «Каталог выгрузки».

Каталог отправки данных отчетности

В поле «Каталог отправки данных отчетности» необходимо указать имя каталога почтового ящика программы электронной почты.

Этот каталог располагается в каталоге программы электронной почты (обычно это каталог C:\DIPOST). Имя каталога отправки данных отчетности состоит из строки EXPT, кода инспекции ФНС, в которую отправляется отчетность, и кода абонента. В этот каталог перед отправкой помещаются фалы с данными форм отчетности.

В общем случае, имя каталога формируется автоматически, и изменять его не следует.

Прочитать данные с установочной дискеты

Перечисленные выше параметры можно ввести путем считывания их со специальной установочной дискеты, выдаваемой каждому абоненту при заключении договора на подключение к системе представления отчетности по каналам связи. Для чтения параметров дискету следует поместить в дисковод и нажать эту кнопку.
