Настройки отчета

Форма настроек отчета вызывается из форм регламентированных отчетов, состоящих из множества разделов (например, это почти все налоговые декларации).

Форма предназначена для:

  • управления автоматической нумерацией страниц отчета;
  • управления порядком заполнения тех полей отчета, значения которых должны выбираться из заранее определенного набора значений;
  • управления автоматическим сохранением отчета;
  • управления показом и печатью разделов отчета.

«Отключить счетчик страниц» - если флажок установлен, номера страниц отчета не будут вычисляться автоматически при формировании печатной формы.

«Отключить выбор значений из списка» - установка этого флажка позволяет отключить вывод на экран списка допустимых значений полей налоговых деклараций, для которых предусмотрены такие списка, и указать значение вручную.

«Автосохранение каждые … минут» - если флажок установлен, при редактировании регламентированного отчета он будет автоматически сохраняться с указанным интервалом времени. Для новых отчетов автоматическое сохранение включается после того, как отчет будет записан в журнале отчетов. Также автоматическое сохранение не будет производиться в следующих случаях:

- если в процессе редактирования отчета был изменен вид документа и/или номер корректировки и после этого данные не были сохранены;

- если в процессе редактирования отчета был изменен код ИФНС (если такой реквизит присутствует на форме), в которую представляется отчет, и после этого данные не были сохранены.

«Проверять соотношения показателей при печати и выгрузке» - если флажок установлен, то перед печатью и выгрузкой будет автоматически запускаться проверка показателей отчета.

Список разделов отчета - в списке можно указать, какие из разделов отчета следует показывать на экране и выводить на печать. Разделы для показа и вывода на печать можно указывать независимо: например, раздел можно заполнить и спрятать, но при печати он будет обязательно напечатан.
