Как добавить типы цен номенклатуры

  1. Откройте список типов цен номенклатуры (Предприятие – Товары (материалы, продукция, услуги) – Типы цен номенклатуры).
  2. Добавьте новый элемент (меню Действия – Добавить).
  3. Заполните поле Наименование – указывается тип цены и организация, для которой заводится этот тип (например, Оптовая – Организация 1).
  4. В поле Валюта цены по умолчанию укажите валюту.
  5. Заполните поля Метод округления и Округлять до.
  6. Сохраните созданный элемент Тип цен номенклатуры – кнопка OK.
