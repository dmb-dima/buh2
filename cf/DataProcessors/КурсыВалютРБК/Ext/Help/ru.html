Загрузка курсов валют

Обработка предназначена для загрузки актуальных курсов валют за определенный период времени.

Обработка вызывается из справочника Валюты по кнопке Загрузить курсы.

После указания периода, загрузка курсов производится по кнопке Загрузить. Загрузка производится с сайта "РосБизнесКонсалтинг" (РБК). После успешной загрузки информация о курсах валют автоматически записывается в регистр Курсы валют.
