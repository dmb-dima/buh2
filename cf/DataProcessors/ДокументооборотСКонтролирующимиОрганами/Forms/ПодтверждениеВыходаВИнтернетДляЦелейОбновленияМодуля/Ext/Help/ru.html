Оповещение об ошибке

Форма предназначена для выдачи предупреждений о наличии ошибок в регламентированной отчетности. В критических случаях, работа отдельных функций формы отчетности или формы отчетности в целом может быть заблокирована.

Оповещения об этих событиях выдаются в виде сообщеним следующих видов:

Не рекомендуется использовать функцию <описание режима формы отчетности> для данной формы регламентированной отчетности

Сообщение выдается в том случае, если для указанной редакции формы отчетности выявлена ошибка (ошибки) в реализации функции, указанной в сообщении, и эта ошибка в определенный ситуациях может привести к подготовке некорректной формы отчетности.

В качестве дополнительной информации приводится подробное описание ошибки (ошибок).

Функция <описание режима формы отчетности> для данной формы регламентированной отчетности заблокирована

Сообщение выдается в том случае, если для указанной редакции формы отчетности выявлена ошибка (ошибки) в реализации функции, указанной в сообщении, и эта ошибка приводит к подготовке некорректной формы отчетности, в связи с чем дальнейшее использование формы отчетности невозможно.

В качестве дополнительной информации приводится подробное описание ошибки (ошибок).

Продолжить действие, несмотря на предупреждение

Флажок присутствует в форме оповещения в том случае, если дальнейшая работа с формой отчетности допускается, но пользователь должен сам принять решение о этом. Чтобы продолжить работу, следует установить флажок и нажать кнопку "Продолжить".

Проверить обновления

Нажатие это кнопки открывается форму со списком доступных для данной версии типовой конфигурации обновлений. Если к моменту выдачи сообщения уже выпущено обновление, содержащее исправление формы отчетности, информация об этом обновлении будет показана в форме.

Из формы с информацией об обновлениях можно перейти либо к форме обновления конфигурации через Интернет, либо на страницу конфигурации на сайте поддержки пользователей программ системы "1С:Предприятие 8" http://users.v8.1c.ru/ для скачивания обновленной формы отчетности.
