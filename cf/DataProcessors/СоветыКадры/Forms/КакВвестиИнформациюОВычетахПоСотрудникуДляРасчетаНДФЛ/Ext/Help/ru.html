Как ввести информацию о стандартных вычетах по сотруднику для расчета НДФЛ

  1. Откройте форму элемента справочника Физические лица (Кадры – Физические лица).
  2. По кнопке НДФЛ введите информацию о вычетах:
    • На закладке Вычеты добавьте новую строку в табличной части раздела Право на личный стандартный вычет (кнопка Добавить в командной панели раздела).
    • В колонке Дата укажите дату, начиная с которой будут применяться стандартные вычеты.
    • В колонке Код укажите код вычета для данного сотрудника (статья 218 НК РФ).
    • Добавьте новую строку в табличной части раздела Право на стандартные вычеты на детей (кнопка Добавить в командной панели раздела).
    • В колонке Период (с ... по) укажите период, в котором будут применяться вычеты.
    • В колонке Код и количество детей укажите код вычета (статья 218 НК РФ), исходя из которого будет рассчитан размер вычета, и количество детей сотрудника. 
  3. Добавьте новую строку в табличной части раздела Применение вычетов (кнопка Добавить в командной панели раздела):
    • В поле Дата укажите дату, начиная с которой будут применяться стандартные вычеты.
    • В поле Применять вычеты в организации укажите организацию, в которую будет принят (или уже принят) данный сотрудник.
  4. Сохраните информацию и закройте форму – кнопка OK.
