Обработка «Управление регламентом установки даты актуальности учета»

Данная обработка позволяет создать и сконфигурировать регламентные задания для автоматической установки даты актуальности. Такие задания могут быть сконфигурированы как для всех организаций в целом, так и для каждой организации в отдельности.

В случае использования автоматической установки даты актуальности регламентными заданиями, дата актуальности будет сдвигаться на текущую дату (с перепроведением всех необходимых документов) по заданному расписанию.

Механизм автоматической установки даты запрета рекомендуется применять в информационных базах с большим количеством вводимых документов, при наличии специально выделенных серверов, работающих в круглосуточном режиме.
