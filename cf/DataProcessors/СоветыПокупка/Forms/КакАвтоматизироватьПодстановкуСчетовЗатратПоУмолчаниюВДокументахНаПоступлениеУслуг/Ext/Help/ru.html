Как настроить подстановку счетов затрат по умолчанию при оформлении услуг сторонних организаций

Для того, чтобы при оформлении услуг сторонних организаций счета затрат по умолчанию подставлялись в документы конфигурации, нужно выполнить следующее:

  1. Открыть регистр сведений Счета учета номенклатуры (Предприятие – Товары (материалы, продукция, услуги) – Счета учета номенклатуры).
  2. Добавить новую запись регистра, нажав в командной панели кнопку Добавить.
  3. В открывшейся форме:
    • поле Номенклатура заполняется значениями из справочника Номенклатура, который открывается при нажатии кнопки Выбрать, расположенной рядом с полем. В качестве значения поля Номенклатура из справочника можно выбрать как целую группу номенклатуры, так и отдельный элемент. Если в справочнике Номенклатура нет нужного значения, то его необходимо добавить.

      Примечание. В справочнике Номенклатура для удобства отдельные элементы можно объединять в группы, например для услуг: Услуги для администативно-управленческого персонала (счет 26), Услуги для производственных цехов (счет 25), и т.д. Добавить группу можно с помощью кнопки Добавить группу, затем в созданную группу нужно добавить элементы.

      При этом следует учитывать, что если в регистре сведений Счета учета номенклатуры настройки будут сделаны для всей группы, то при выборе отдельного элемента группы при оформлении услуг сторонних организаций будут подставляться значения, указанные для всей группы.

        • Поле Счет учета – выберите счет затрат из Плана счетов (например, сч.26).
        • Поле Статьи затрат – выберите статью затрат.
        • В поле Счет учета доходов от реализации – сч. 90.01.1.
        • В поле Счет учета предъявленного НДС – сч. 19.04.
        • В поле Счет учета НДС по реализации – сч. 90.03.

      4. Для сохранения введенной информации нажмите кнопку OK.
