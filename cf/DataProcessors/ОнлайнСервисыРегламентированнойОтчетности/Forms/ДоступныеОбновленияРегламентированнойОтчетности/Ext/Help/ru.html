Информация об обновлениях

Форма содержит сведения об обновлениях регламентированной отчетности, выпущенных в виде новых версий конфигурации или дополнении к текущей версии.

Обновления конфигураций

Раздел содержит информацию о последнем выпущенном релизе конфигурации, а также позволяет открыть описание нового в релизе в части регламентированной отчетности.

Перейти к обновлению...

Нажатие кнопки открывает форму для автоматического обновления конфигурации через Интернет.

Если между текущим релизом конфигурации и релизом, номер которого отражен в информации об обновлениях, выпущены более одного релиза, для обновления текущего релиза на последний выпущенный необходимо запускать форму автоматического обновления конфигурации через Интернет последовательно несколько раз.

Форма с информацией о обновлениях не содержит информацию о новых редакциях текущей конфигурации.

Показать отличия от текущей версии

Щелчок по надписи открывается описание отличий последнего выпущенного релиза (номер которого отражен в информации об обновлениях) от текущего релиза в части обновления регламентированной отчетности. Полное описание новшеств последнего выпущенного релиза конфигурации можно посмотреть на странице конфигурации на сайте поддержки пользователей системы "1С:Предприятия 8" http://users.v8.1c.ru/.

Дополнения для текущей версии конфигурации

Раздел содержит информацию об обновлениях форм регламентированной, которые были выпущены для текущей версии конфигурации как для целей оперативного отражения в конфигурации изменений законодательства в части подготовки и представления регламентированной отчетности, так и для целей исправления выявленных ошибок.

Список содержит доступные обновления. Для получения обновлений необходимо перейти на страницу конфигурации на сайте поддержки пользователей системы «1С:Предприятия 8» http://users.v8.1c.ru/ . Сделать это можно, щелкнув мышью надпись "Открыть Web-страницу текущей версии".
