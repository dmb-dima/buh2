Настройки оповещений

Форма предназначена для управления функциями оповещения пользователя о ситуациях, способных привести к ошибкам в подготовленной регламентированной отчетности и, как следствие, к возможным проблемам при представлении отчетности в контролирующие органы.

Общие настройки

Параметры этой группы регулируются пользователем, имеющим полные права, и задаются для всей информационной базы в целом.

Выполнять проверки регламентированной отчетности

При установленном флажке программа будет выполнять проверки и оповещать пользователей регламентированной отчетности:

  • об окончании действия редакций форм регламентированной отчетности и версий форматов электронного представления форм отчетности в связи в вступлением в силу новых редакций и версий;
  • о наличии ошибок в регламентированной отчетности и, в критических случаях, блокировка отдельных функций или формы отчетности в целом;
  • о выходе новых релизов конфигураций или выпуске дополнений к текущим релизам.

используя для этого служебную информацию, получаемую с сервера фирмы "1С" через Интернет.

Уведомлять пользователей об ошибках получения сведения из Интернет

При установленном флажке пользователь, с компьютера которого выполняется доступ в Интернет для получения служебной информации, будет получать сообщения об ошибках.

Индивидуальные настройки

Параметры этой группы регулируются каждым пользователем индивидуально и задаются только для конкретного рабочего места.

Разрешить программе получение информации с сервера фирмы "1С" через Интернет

На конкретном рабочем месте должно быть указано: будет ли именно на этом рабочем месте выполняться скачивание через Интернет служебной информации, необходимой для работы функций оповещения. Установкой этого флажка пользователь должен подтвердить, что он разрешает программе доступ в Интернет со своего компьютера.

При установленном флажке каждый раз при начале работы пользователя с регламентированной отчетностью в форме "Регламентированная и финансовая отчетность", программа будет выполнять обновление через Интернет с сервера фирмы "1С" служебной информации.

Обновленная информация будет доступна всем пользователям этой информационной базы, работающим с регламентированной отчетностью.

Для доступа в Интернет необходима авторизация на прокси-сервере

Если для доступа в Интернет должны быть заданы параметры авторизации, установите этот флажок и укажите имя пользователя и пароль. Их следует получить у системного администратора.
