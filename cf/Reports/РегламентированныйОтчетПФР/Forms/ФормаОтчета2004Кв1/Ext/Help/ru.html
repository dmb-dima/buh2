Отчет «Декларация в ПФР»

Отчет «Декларация в ПФР» преданзначен для подготовки налоговой декларации по страховым взносам на обязательное пенсионное страхование для лиц, производящих выплаты физическим лицам: организаций, индивидуальных предпринимателей; физических лиц, не признаваемых индивидуальными предпринимателями, по форме, утвержденной приказом Минфина России от 24.01.2005 г. №9н.

Регламентированный отчет по выбранной пользователем форме открывается после нажатия кнопки «ОК» в «стартовой» форме отчета. В открытом отчете будет установлен отчетный период, указанный в стартовой форм. Изменить отчетный период в уже открытом отчете нельзя: если отчет не записан, можно отказаться от ввода отчета и, вновь через стартовую форму, открыть отчет за другой период.

Если отчет допускает автоматческое заполнение, в верхней командной панели отчета будет присутстововать кнопка "Заполнить".

Если такой возможности не предусмотрено, отчет следует заполнять вручную.

Порядок заполнения

При заполнении декларации рекомендуется соблюдать такую последовательность подготовки листов

1.  Титульный лист.
2.  Сведения о физическом лице (вторая страница титульного листа).
3.  Раздел 2.2 «Расчет для заполнения строк 0100, 0300 и 0400».
4. Раздел 2.3 «Сводные показатели за расчетный период для расчета сумм платежей по страховым взносам на обязательное пенсионное страхование организациями, имеющими в своем составе обособленные подразделения, исполняющие обязанности организаций по уплате страховых взносов и представлению отчетности».
5.  Раздела 2.4 «Расчет условия на право применения регрессивных тарифов страховых взносов на обязательное пенсионное страхование по месяцам последнего квартала расчетного периода (заполняется  страхователями, применяющими специальные налоговые режимы, включая осуществляющих одновременно виды предпринимательской деятельности, облагаемые по общему налоговому режиму)».
6.  Раздел 2 «Расчет платежей по страховым взносам на обязательное пенсионное страхование».
7.  Раздел 2.1 «Уплата платежей по страховым взносам на обязательное пенсионное страхование».
8.  Раздел 1 «Платежи по страховым взносам на обязательное пенсионное страхование, подлежащие уплате, по данным страхователя».

Примечания

Разделы 2.3 и 2.4 могут быть заполнены в произвольной последовательности.

Страница первая титульного листа, разделы 1, 2, 2.1, 2.2 декларации по страховым взносам представляются всеми страхователями. Страница вторая титульного листа представляется физическим лицом, не признаваемым индивидуальным предпринимателем, в случае отсутствия у него ИНН.

Раздел 2.3. заполняется организациями, имеющими обособленные подразделения, исполняющие обязанности организации по исчислению и уплате страховых взносов и представлению отчетности по страховым взносам. Представляется головной организацией в налоговый орган по месту ее постановки на учет.

Раздел 2.4 заполняется страхователями:
 - уплачивающими единый налог на вмененный доход для отдельных видов деятельности;
 - применяющими упрощенную систему налогообложения;
 - применяющими систему налогообложения для сельскохозяйственных товаропроизводителей.

Особенности заполнения некоторых листов декларации

Титульный лист

Следует указать сведения об организации и сведения о должностных лицах, которые не были заполнены автоматически. Автоматически заполненные сведения необходимо проверить и, при необходимости, отредактировать.

Сведения о физическом лице (вторая страница титульного листа)

Заполняется только в отношении физического лица, не признаваемого индивидуальным предпринимателем, у которого отсутствует ИНН.
Автоматического заполнение листа в данной декларации не предусмотрено.

Раздел 1. «Платежи по страховым взносам на обязательное пенсионное страхование, подлежащие уплате, по данным страхователя»

Суммы платежей по страховым взносам, подлежащие уплате в ПФР на страховую часть трудовой пенсии по месяцам последнего квартала отчетного периода, переносятся из графы 6 строк 0320 - 0340 Раздела 2 декларации по страховым взносам соответственно.

Суммы платежей по страховым взносам, подлежащие уплате в ПФР на накопительную часть трудовой пенсии по месяцам последнего квартала отчетного периода, переносятся из графы 6 строк 0420 - 0440 Раздела 2 декларации по страховым взносам соответственно.

Раздел 2. «Расчет платежей по страховым взносам на обязательное пенсионное страхование»

База для начисления страховых взносов за отчетный период, отражаемая по строке 0100, переносится из Раздела 2.2. «Расчет для заполнения строк 0100, 0300 и 0400» :
 по графе 3 - из графы 4 строки 500;
 по графе 4 - из графы 4 строки 501;
 по графе 5 -  из графы 4 строки 502;
 по графе 6 - сумма строк 500, 501, 502 по графе 4;
 по графе 7 - сумма строк 500, 501, 502 по графе 5.   
 
Сумма начисленных платежей по страховым взносам на страховую часть трудовой пенсии, отражаемая по строке 0300, переносится из Раздела 2.2  «Расчет для заполнения строк 0100, 0300 и 0400» :
 по графе 3 - из графы 7 строки 500;
 по графе 4 - из графы 7 строки 501;
 по графе 5 - из графы 7 строки 502;
 по графе 6 - сумма строк 500, 501, 502 по графе 7.    
 
Сумма начисленных платежей по страховым взносам на накопительную часть трудовой пенсии, отражаемая по строке 0400, переносится из Раздела 2.2  «Расчет для заполнения строк 0100, 0300 и 0400» :
 по графе 4 - из графы 9 строки 501;
 по графе 5 - из графы 9 строки 502;
 по графе 6 - сумма строк 501 и 502 по графе 9.

Строки 0300 - 0340, 0400 - 0440 по графе 7 заполняются страхователями, являющимися плательщиками единого налога на вмененный доход для отдельных видов деятельности. По этим строкам должны отражаться данные только в части застрахованных лиц, занятых в сферах деятельности, не переведенных на уплату  единого налога на вмененный доход.

Суммы начисленных платежей по страховым взносам на обязательное пенсионное страхование за отчетный период, отражаемые по строкам 0200 - 0240 в графе 6, должны соответствовать данным строк 0300 - 0340 по графе 3  декларации по единому  социальному налогу.

Раздел 2.1. «Уплата платежей по страховым взносам на обязательное пенсионное страхование»

Разница между суммами платежей по страховым взносам, начисленными за соответствующий период, и суммами страховых взносов, уплаченными за тот же период, отражаемая по строкам 015 - 019, определяется как:
 строка 015 графа 3 = строка 0300 графа 3 декларации по ЕСН - строка 010 графа 3;
 строка 016 графа 3 = строка 0310 графа 3 декларации по ЕСН - строка 011 графа 3;
 строка 017 графа 3 = строка 0320 графа 3 декларации по ЕСН - строка 012 графа 3;
 строка 018 графа 3 = строка 0330 графа 3 декларации по ЕСН - строка 013 графа 3;
 строка 019 графа 3 = строка 0340 графа 3 декларации по ЕСН - строка 014 графа 3;
 строка 015 графа 4 = строка 0300 графа 3 декларации по ЕСН - строка 010 графа 4;
 строка 016 графа 4 = строка 0310 графа 3 декларации по ЕСН - строка 011 графа 4;
 строка 017 графа 4 = строка 0320 графа 3 декларации по ЕСН - строка 012 графа 4;
 строка 018 графа 4 = строка 0330 графа 3 декларации по ЕСН - строка 013 графа 4;
 строка 019 графа 4 = строка 0340 графа 3 декларации по ЕСН - строка 014 графа 4.

По строкам 015 - 019 учитывается положительная разница между начисленными и уплаченными суммами страховых взносов на обязательное пенсионное страхование.
