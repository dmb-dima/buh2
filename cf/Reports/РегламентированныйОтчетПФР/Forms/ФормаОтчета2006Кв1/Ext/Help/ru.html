Отчет «Декларация в ПФР»

Отчет «Декларация в ПФР» преданзначен для подготовки налоговой декларации по страховым взносам на обязательное пенсионное страхование для лиц, производящих выплаты физическим лицам: организаций, индивидуальных предпринимателей; физических лиц, не признаваемых индивидуальными предпринимателями, по форме, утвержденной приказом Минфина России от 27.02.2006 № 30н.

Регламентированный отчет по выбранной пользователем форме открывается после нажатия кнопки «ОК» в «стартовой» форме отчета. В открытом отчете будет установлен отчетный период, указанный в стартовой форме. Изменить отчетный период в уже открытом отчете нельзя: если отчет не записан, можно отказаться от ввода отчета и, вновь через стартовую форму, открыть отчет за другой период.

Если отчет допускает автоматческое заполнение, в верхней командной панели отчета будет присутстововать кнопка "Заполнить".

Если такой возможности не предусмотрено, отчет следует заполнять вручную.

Порядок заполнения

При заполнении декларации рекомендуется соблюдать такую последовательность подготовки листов

1.  Титульный лист.
2.  Раздел 2.2 «Расчет для заполнения строки 0100».
4. Раздел 2.3 «Сводные показатели за расчетный период для расчета сумм платежей по страховым взносам на обязательное пенсионное страхование организациями, имеющими в своем составе обособленные подразделения, исполняющие обязанности организаций по уплате страховых взносов и представлению декларации».
5.  Раздел 2 «Расчет платежей по страховым взносам на обязательное пенсионное страхование».
6.  Раздел 2.1 «Уплата платежей по страховым взносам на обязательное пенсионное страхование».
7.  Раздел 1 «Платежи по страховым взносам на обязательное пенсионное страхование, подлежащие уплате».

Примечания

Страница первая титульного листа, разделы 1, 2, 2.1, 2.2 декларации по страховым взносам представляются всеми страхователями. Страница вторая титульного листа представляется физическим лицом, не признаваемым индивидуальным предпринимателем, в случае отсутствия у него ИНН. По умолчанию данный лист декларации всегда скрыт и на печать не выводится. В случае необходимости работы с данным листом, следует нажать на кнопку «Настройка» и в появившейся форме установить параметры для листа, содержащего сведения о физ. лицах.

Раздел 2.3. заполняется организациями, имеющими обособленные подразделения, исполняющие обязанности организации по исчислению и уплате страховых взносов и представлению отчетности по страховым взносам. Представляется головной организацией в налоговый орган по месту ее постановки на учет.

Особенности заполнения некоторых листов декларации

Титульный лист

Автоматически заполненные сведения необходимо проверить и, при необходимости, отредактировать.

Сведения о физическом лице (вторая страница титульного листа)

Заполняется только в отношении физического лица, не признаваемого индивидуальным предпринимателем, у которого отсутствует ИНН.
Автоматического заполнение листа в данной декларации не предусмотрено. По умолчанию данный лист не отображается в основной панели и не выводится на печать.

Раздел 1. «Платежи по страховым взносам на обязательное пенсионное страхование, подлежащие уплате».

Суммы платежей по страховым взносам, подлежащие уплате в ПФР на страховую часть трудовой пенсии по месяцам последнего квартала расчетного периода, переносятся из графы 5 строк 0300 - 0340 Раздела 2 декларации по страховым взносам соответственно.
                           
Суммы платежей по страховым взносам, подлежащие уплате в ПФР на накопительную часть трудовой пенсии по месяцам последнего квартала расчетного периода, переносятся из графы 5 строк 0400 - 0440 Раздела 2 декларации по страховым взносам соответственно.

Раздел 2. «Расчет платежей по страховым взносам на обязательное пенсионное страхование»

База для начисления страховых взносов за отчетный период, отражаемая по строке 0100, переносится из Раздела 2.2. «Расчет для заполнения строки 0100» :
 по графе 3 - из графы 4 строки 400;
 по графе 4 - из графы 4 строки 401;
 по графе 5 -  из графы 4 строки 402;
 по графе 6 - из графы 5 строки 402; 

Раздел 2.1. «Уплата платежей по страховым взносам на обязательное пенсионное страхование»

Значения показателей по строкам 015 - 019 определяются, соответственно, как:
стр. 015 гр. 3 (4) = стр. 0300 гр. 3 декларации по ЕСН - стр. 010 гр. 3 (4);
стр. 016 гр. 3 (4) = стр. 0310 гр. 3 декларации по ЕСН - стр. 011 гр. 3 (4);
стр. 017 гр. 3 (4) = стр. 0320 гр. 3 декларации по ЕСН - стр. 012 гр. 3 (4);
стр. 018 гр. 3 (4) = стр. 0330 гр. 3 декларации по ЕСН - стр. 013 гр. 3 (4);
стр. 019 гр. 3 (4) = стр. 0340 гр. 3 декларации по ЕСН - стр. 014 гр. 3 (4).

В целях реализации абзаца четвертого пункта 3 статьи 243 Кодекса учитывается положительная разница между начисленными и уплаченными суммами страховых взносов на обязательное пенсионное страхование, отражаемая по строкам 015 - 019.
