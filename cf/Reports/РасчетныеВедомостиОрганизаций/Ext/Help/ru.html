Отчет "Расчетная ведомость организации"

Зарплата ® Расчетная ведомость (произвольная форма)

Отчет предназначен для формирования расчетной ведомости произвольной формы для внутренней отчетности организации.

Отбор по организации или подразделению можно задать по кнопке Отбор.

Формирование более компактного вида отчета со скрытым заголовком выполняется по кнопке Заголовок.
