Отчет "Регистр учета доходов текущего периода"

Отчеты ® Регистры налогового учета по налогу на прибыль ® Регистр учета доходов текущего периода

Регистр предназначен для обобщения информации об операциях получения доходов отчетного (налогового) периода с целью выявления сумм доходов, в том числе внереализационных, используемых при заполнении декларации по налогу на прибыль организаций.

Параметры формирования отчета можно задать по кнопке Настройка. Например, отбор по виду доходов.
