Отчет «Налог на игорный бизнес»

Отчет «Налог на игорный бизнес» предназначен для подготовки декларации по налогу на игорный бизнес в соответствии с Приложением №1 к приказу Министерства финансов Российской Федерации от 1 ноября 2004 г. № 97н.

После вызова формы декларации из списка «Регламентированные отчеты», в диалоге управления отчетом следует, в первую очередь, указать организацию, для которой будет заполняться отчет.

Порядок заполнения

Рекомендуется соблюдать такую последовательность подготовки листов:

1.   Титульный лист.

2.   Раздел 3.1 «Изменение количества игровых столов за налоговый период».

3.   Раздел 3 «Изменение количества объектов налогообложения налогом на игорный бизнес за налоговый период».

4.   Раздел 2.1 «Расчет налога на игорный бизнес по игровым столам».

5.   Раздел 2 «Расчет налога на игорный бизнес».

6.   Раздел 1 «Сумма налога, подлежащая уплате в бюджет, по данным налогоплательщика».

Особенности заполнения некоторых листов декларации

Титульный лист

Следует указать сведения об организации и сведения о должностных лицах, которые не были заполнены автоматически. Автоматически заполненные сведения необходимо проверить и, при необходимости, отредактировать.

Раздел 3.1 «Изменение количества игровых столов за налоговый период»

При заполнении данных по разделу 3.1 следует иметь в виду, что при наличии игровых столов с различным количеством игровых полей раздел 3.1 заполняется столько раз, сколько имеется вариантов количества игровых полей в игровых столах. При этом  данные по всем игровым столам, имеющим одинаковое количество игровых полей, заполняются на одной странице разделе 3.1.

Для создания дополнительных страниц по разделу 3.1 следует воспользоваться механизмом создания дополнительных листов, органы управления которого расположены в диалоге отчета. При нажатии кнопки «Добавить» добавляется новый бланк (вариант заполнения) раздела, при нажатии кнопки «Удалить» удаляется текущий бланк, а нажатие на кнопки «<» и «>» приводит к переключению между бланками.

Раздел 3 «Изменение количества объектов налогообложения налогом на игорный бизнес за налоговый период»

При заполнении раздела 3 необходимо учесть, что данные по графе 3 (по игровым столам) переносятся автоматически из раздела 3.1 «Изменение количества игровых столов за налоговый период» как сумма соответствующих показателей по всем по игровым столам с различным количеством игровых полей (дополнительным листам) раздела 3.1.

Раздел 2.1 «Расчет налога на игорный бизнес по игровым столам»

Количество игровых столов по коду строки 040 определяется автоматически как сумма строк 020, 030 и 050 по гр. 3 раздела 3.1 по игровым столам,  имеющим соответствующее количество игровых полей, указанное в строке 010 графы 3.

При заполнении данных по разделу 2.1 следует иметь в виду, что при наличии игровых столов с различным количеством игровых полей раздел 2.1 заполняется столько раз, сколько имеется вариантов количества игровых полей в игровых столах. При этом данные по всем игровым столам, имеющим одинаковое количество игровых полей, заполняются на одной странице разделе 2.1.

Для создания дополнительных страниц по разделу 2.1 следует воспользоваться механизмом создания дополнительных листов, органы управления которого расположены в диалоге отчета.

Раздел 2 «Расчет налога на игорный бизнес»

При заполнении раздела 2 необходимо учесть, что данные по графе 3 (по игровым столам) переносятся автоматически из раздела 2.1 «Расчет налога на игорный бизнес по игровым столам» как сумма соответствующих показателей по всем игровым столам с различным количеством игровых полей (дополнительным листам) раздела 2.1.

Общее количество  соответствующих объектов игорного бизнеса по коду строки 020 по графам 4, 5 и 6 определяется автоматически как сумма строк 010, 020 и 040 раздела 3 по соответствующим графам.

Раздел 1 «Сумма налога, подлежащая уплате в бюджет, по данным налогоплательщика»

Коды (код) бюджетной классификации, предлагаемые для выбора при заполнении строки «Код бюджетной классификации», зависят от даты подписания отчета,
указанной в диалоге управления титульного листа декларации.

Если в качестве даты подписания декларации указана дата, включая или позже 1 января 2005 года, предполагается, что декларация представляется в 2005 году и налог будет уплачиваться также в 2005 году, поэтому при вводе КБК будут выданы для выбора 20-значные коды.

Если в качестве даты подписания указана дата, более ранняя, чем 1 января 2005 года, для выбора будут выданы «старые» 7-значные коды.

Сумма налога на игорный бизнес, подлежащая  уплате в бюджет и отражаемая в строке по коду 030, определяемая как сумма значений граф 3,  4,  5  и 6 строки 070 раздела 2 налоговой декларации.
