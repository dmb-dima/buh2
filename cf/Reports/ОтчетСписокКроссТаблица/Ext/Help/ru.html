Отчет "Список/Кросс-таблица"

Отчеты ® Прочие ® Список/Кросс-таблица

Отчет предназначен для получения списков и кросс-таблиц. Кросс-таблица представляет собой отчет с группировкой данных в строках и в колонках.

Отчет может быть построен для любого регистра накопления конфигурации.

По регистру накопления остатков может быть построен отчет, выводящий остатки на заданную дату.

По регистру накопления оборотов может быть построен отчет, выводящий оборот за заданный период. Оборот может быть дополнительно детализирован по годам, месяцам, неделям, дням.

Построение отчета выполняется по кнопке Сформировать.

В поле Раздел учета может быть выбран любой регистр накопления конфигурации. При выборе раздела учета выполняется начальная настройка отчета.

Для сохранения настроек отчета для использования в дальнейшем предназначены кнопки сохранения и восстановления значений.

Параметры формирования отчета можно задать по кнопке Настройка.

  • На закладке Общее указываются основные параметры отчета: использовать свойства и категории, раскрашивать группировки, показатели, выводить в разных колонках.
  • На закладке Группировки задается список полей, по которым необходимо группировать данные. Каждая строка списка определяет поле группировки и тип итогов.
  • На закладке Поля задается список выводимых в отчет полей.
  • На закладке Отбор задается отбор данных диаграммы.
  • На закладке Сортировка задается упорядок формирования строк отчета.
