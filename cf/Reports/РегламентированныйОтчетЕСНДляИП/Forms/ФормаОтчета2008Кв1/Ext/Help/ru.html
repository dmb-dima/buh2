Отчет «Декларация по ЕСН для ИП»

Отчет «Декларация по ЕСН для ИП» предназначен для подготовки налоговой декларации по единому социальному налогу для индивидуальных предпринимателей, глав крестьянских (фермерских) хозяйств, адвокатов по форме, утверждённой приказом Минфина России от 17.12.2007 № 132н.

Регламентированный отчет по выбранной пользователем форме открывается после нажатия кнопки «ОК» в «стартовой» форме отчета. В открытом отчете будет установлен отчетный период, указанный в стартовой форме. Изменить отчетный период в уже открытом отчете нельзя: если отчет не записан, можно отказаться от ввода отчета и, вновь через стартовую форму, открыть отчет за другой период.

Перед началом работы по составлению регламентированных отчетов рекомендуется ознакомиться с описанием работы с регламентированными отчетами.

Порядок заполнения

Для правильного переноса данных между листами декларации, рекомендуется соблюдать такую последовательность подготовки листов:

1. Титульный лист.

2. Раздел 2 "Расчет единого социального налога за налоговый период" .

3. Раздел 1 "Сумма налога, подлежащая доплате (уменьшению) в федеральный бюджет и фонды обязательного медицинского страхования, по данным налогоплательщика".

Особенности заполнения некоторых листов декларации

Титульный лист

Следует указать сведения об организации и сведения о должностных лицах, которые не были заполнены автоматически. Автоматически заполненные сведения необходимо проверить и, при необходимости, отредактировать.

Раздел 1  "Сумма налога, подлежащая доплате (уменьшению) в федеральный бюджет и фонды обязательного медицинского страхования, по данным налогоплательщика"

При заполнении Раздела 1 декларации налогоплательщиком  по строке 001 указывается:

03 - для индивидуальных предпринимателей, облагаемых по общему налоговому режиму;

13 - для индивидуальных предпринимателей, облагаемых по общему налоговому режиму и одновременно применяющих специальные налоговые режимы;

06 - для глав крестьянских (фермерских) хозяйств, облагаемых по общему налоговому режиму;

16 - для глав крестьянских (фермерских) хозяйств, облагаемых по общему налоговому режиму и одновременно применяющих специальные налоговые режимы; 

07 - для адвокатов, осуществляющих профессиональную деятельность в коллегиях адвокатов, адвокатских бюро и юридических консультациях; 

08 - для адвокатов, учредивших адвокатские кабинеты;

09 - для нотариусов, занимающихся частной практикой.

По строке 010 указывается код административно-территориального образования в соответствии с Общероссийским классификатором административно-территориального деления ОК 019-95 (ОКАТО).

По строкам 030 указываются суммы налога, подлежащие доплате (+) (уменьшению (-)) в федеральный бюджет и (или) фонды обязательного медицинского страхования по итогам налогового периода. Суммы налога автоматически переносятся из соответствующих строк раздела 2.

Раздел 2 "Расчет единого социального налога за налоговый период"

В Разделе 2 налогоплательщиком отражается информация о налоговой базе, сумме налоговых льгот, суммах исчисленного им налога, а также суммах налога, подлежащих доплате (уменьшению) в федеральный бюджет и фонды обязательного медицинского страхования за налоговый период.
