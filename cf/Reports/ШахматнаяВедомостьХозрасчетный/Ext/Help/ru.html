Отчет "Шахматная ведомость"

Отчеты ® Шахматная ведомость

Отчет предназначен для представления оборотов между счетами за выбранный период.

Каждая строка отчета соответствует дебетуемым счетам, каждый столбец - кредитуемым счетам.

Построение отчета выполняется по кнопке Сформировать.

Формирование более компактного вида отчета со скрытым заголовком выполняется по кнопке Заголовок.

В отчет можно вывести данные по валютам, по субсчетам, а также по забалансовым счетам.
