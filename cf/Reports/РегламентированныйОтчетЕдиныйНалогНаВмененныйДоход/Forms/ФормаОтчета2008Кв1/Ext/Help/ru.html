Отчет «Декларация по единому налогу на вмененный доход для отдельных видов деятельности»

Налоговая декларация по единому налогу на вмененный доход для отдельных видов деятельности предназначен для подготовки декларации по единому налогу на вмененный доход для отдельных видов деятельности в соответствии с Приложением №1 к приказу Министерства финансов Российской Федерации от 18 января 2006 г. № 8н (в редакции приказа Минфина РФ от 19.12.2006 № 177н, от 20.09.2007 № 83н).

Если в состав организации входят обособленные подразделения, не имеющие отдельного баланса и расчетного счета, и осуществляющие по месту своего нахождения предпринимательскую деятельность, подпадающую под действие системы налогообложения в виде единого налога на вмененный доход для отдельных видов деятельности, то налоговая декларация должна быть также представлена в налоговый орган по месту осуществления каждой такой деятельности.

Порядок заполнения

Рекомендуется соблюдать такую последовательность подготовки листов:

1. Титульный лист.

2. Раздел 3. Расчет суммы единого налога на вмененный доход за налоговый период.

3. Раздел 2. Расчет единого налога на вмененный доход для отдельных видов деятельности.

4. Раздел 3.1 Сумма единого налога на вмененный доход, подлежащая уплате в бюджет за налоговый период. В данный раздел переносятся показатели из других разделов, поэтому необходимо проверить значения, который автоматически заполнились.

5. Раздел  1 Сумма единого налога на вмененный доход для отдельных видов деятельности, подлежащая уплате в бюджет по данным налогоплательщика.

Особенности заполнения некоторых листов декларации

Титульный лист

Следует проверить все автоматически заполненные сведения об организации и должностных лицах. Автоматически заполненные сведения, при необходимости, можно отредактировать в соответствующих справочниках и регистрах информационной базы. Нажатие кнопки  «Обновить», вызовет повторное заполнение титульного и других листов, где отображаются сведения об организации и ответственных лицах.

Раздел 2.  «Расчет единого налога на вмененный доход для отдельных видов деятельности»

При заполнении раздела 2, показатель в строке 090 «Корректирующий коэффициент К2» с 1 января 2006 года значения корректирующего коэффициента К2 могут устанавливаться в пределах от 0,005 до 1 включительно.

После полного заполнения раздела 2 и раздела 3 можно воспользоваться автозаполнением раздела 3.1 и раздела 1 на основании раздела 2 и раздела 3. Для этого необходимо после заполнения разделов 2 и 3 нажать кнопку «Заполнить». Следует обратить внимание, что уже имеющиеся данные в разделах 1 и 3.1 будут очищены и заполнены заново.

Раздел 3. «Расчет суммы единого налога на вмененный доход за налоговый период»

При заполнении раздела 3, показатель в строке 010 «Сумма исчисленного единого налога на вмененный доход» определяется автоматически при изменении показателя по строке 140 раздела 2 декларации.

Показатели строк 040 и 050 также рассчитываются автоматически.

Раздел 3.1 «Сумма единого налога на вмененный доход, подлежащая уплате в бюджет за налоговый период»

Раздел 3.1 может быть заполнен автоматически, при условии заполненного раздела 2 и 3 декларации.

Раздел можно заполнять используя ручной ввод показателей. В данном разделе все суммовые показатели рассчитываются автоматически получая данные из разделов 2 и 3 декларации.

При вводе кода ОКАТО по строке 010 будет произведен расчет раздела с учетом введенного кода ОКАТО.

Показатель по строке 020 «Налоговая база, исчисленная по всем кодам ОКАТО» заполняется по итогам раздела 2, для всех кодов ОКАТО.

Показатель по строке 030 «Налоговая база, исчисленная по данному коду ОКАТО» заполняется по итогам раздела 2 с учетом кода ОКАТО, который указан в строке 010 данного раздела.

Нельзя ввести более одной страницы с одинаковым кодом ОКАТО.
