Отчет "Анализ налогового учета по УСН"

Отчеты ® Анализ состояния налогового учета по УСН

Отчет предназначен для анализа структуры доходов и расходов налогового учета по УСН.

Отчет состоит из общей схемы налоговой базы и расшифровок отдельных блоков этой схемы.

Переход между отчетами разного уровня осуществляется по кнопке Назад и Структура налоговой базы.
