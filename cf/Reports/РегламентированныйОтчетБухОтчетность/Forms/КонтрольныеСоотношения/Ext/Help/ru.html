Результаты проверки соотношений показателей регламентированных отчетов

Данная форма предназначена для вывода результатов проверки отчета.

Флаг «Отобразить формулы всех периодов» предназначен для показа пользователю всех формул, которые составлены для данного отчета. Формулы могут быть разбиты по периодам. Например, для налогового периода действует один набор формул, а для отчетного периода действует другой набор соотношений. Данного флага может не быть на форме, т.к. для некоторых отчетов предусмотрен только отчетный или только налоговый период.

Флаг «Отобразить только ошибочные соотношения» предназначен для отображения формул, которые вызвали ошибку соотношения. По умолчанию данный флаг всегда установлен. В случае сброса флажка, механизм выведет формулы вне зависимости от результата выполнения проверок соотношения - этот режим полезен тем, что можно увидеть все обработанные формулы к текущему отчету.

Для каждой формулы соответствия подключена форма расшифровки, которая вызывается при двойном клике на строке колонки «Проверяемое соотношение» или колонки «Расшифровка значений».

Форма расшифровки позволяет детально увидеть состав формулы, из каких значений она состоит и из каких регламентированных отчетов значение получено. Форма расшифровка может открывать связанный регламентированный отчет, в том случае, если какой-либо показатель из этого отчета участвует в формуле соотношений.
