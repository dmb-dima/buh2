Настройка расшифровки отдельных показателей бухгалтерской отчетности

Форма настройки расшифровки отдельных показателей бухгалтерской отчетности предназначена для настройки состава строк расшифровки показателей.

Расшифровки отдельных показателей бухгалтерской отчетности бывают двух видов:
• Показатели с фиксированным набором строк расшифровки;
• Показатели с произвольным набором строк расшифровки (выделяются в таблице настройки жирным шрифтом). Для таких показателей допускается указание состава по объектам или видам объектов аналитического учета.

Форма настройки содержит командную панель, на которой размещены кнопки управления списком настройки и сам список настройки имеющий древовидную структуру.

Управления списком настройки осуществляется следующими командами:
«Добавить» (иконка ) – команда позволяет добавить строку к выделенному показателю с произвольным набором строк расшифровки.
«Удалить» (иконка ) – команда удаляет выделенную строку из набора строк показателя с произвольным набором строк расшифровки.
«Отметить все» (иконка ) – команда устанавливает флажки на всех строках расшифровки показателей.
«Снять все отметки» (иконка ) – команда снимает флажки на всех строках расшифровки показателей.
«Сбросить» - команда восстанавливает настройки списка строк расшифровки заданные по умолчанию.

Список настройки состоит из следующих колонок:
«Код строки» - код строки расшифровываемого показателя бухгалтерской отчетности.
«Включать в отчет» - флажок, определяющий нужно или не нужно выводить строку расшифровки в отчет.

При наличии в отчете показателей с произвольным набором строк расшифровки дополнительно отображаются 2 колонки:
«Уровень существ. в %» - параметр, задающий значение в процентах от основного показателя по которому определяется включение строки расшифровки в отчет. Все показатели ниже этого значения в отчет не включаются.
«Макс. кол-во строк» - параметр определяет максимальное количество строк, которое будет включено в отчет для показателей с произвольным набором строк расшифровки. Отбор показателей осуществляется в порядке их существенности.
В случае если для таких показателей задан набор строк, то эти значения при заполнении отчета не будут учитываться.

Примеры настройки

1. Включение в состав отчета фиксированной строки расшифровки показателя.
Для этого достаточно установить флажок на соответствующей строке списка настройки

2. Включение в состав отчета расшифровки показателя с произвольным набором строк расшифровки по заданным критериям на основании сведений информационной базы.
Строки настройки расшифровки таких показателей отображаются жирным шрифтом. Для того чтобы ограничить количество строк расшифровки в отчете следует задать уровень существенности и максимальное количество строк.
Если задать уровень существенности равным 0%, то в отчет будут попадать строки расшифровки, ограничиваемые лишь заданным максимальным количеством строк  в порядке убывания уровня существенности.
Если задать максимальное количество строк равным 0, то в отчет будут попадать все строки расшифровки, имеющие уровень существенности не ниже заданного.
Установка обоих параметров в нулевые значения позволит включить в отчет все строки расшифровки с ненулевыми показателями.

3. Включение в состав отчета расшифровки показателя по заданному списку объектов или видов объектов аналитического учета.
Этот способ настройки доступен только для показателей с произвольным набором строк расшифровки. Если для такого показателя задан список объектов или видов объектов аналитического учета, то значения существенности и максимального количества строк не играют роли при отборе строк. В отчет будут выведены все строки расшифровки по заданному списку, независимо от значения показателей расшифровки.
