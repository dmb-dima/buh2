Отчет «Налоговая декларация по транспортному налогу»

Отчет предназначен для подготовки декларации по транспортному налогу в соответствии с приказом Минфина РФ от 13.04.2006 № 65н.

После вызова формы отчета из списка «Регламентированные отчеты», в диалоге управления отчетом следует, в первую очередь, указать организацию, для которой будет заполняться отчет.

В качестве даты подписи отчета при первом открытии отчета ставится рабочая дата. В зависимости от даты подписания отчета, при вводе кодов бюджетной классификации в тех разделах отчета, где это предусмотрено, для выбора будут выдаваться 20-значные КБК действующие в год, который относится к дате подписания отчета.

Порядок заполнения

При заполнении декларации  рекомендуется соблюдать такую последовательность подготовки листов:

1.Титульный лист.

3. Раздел 2 «Расчет суммы налога по каждому транспортному средству (по данным налогоплательщика)».

2. Раздел 1 «Сумма налога, подлежащая уплате в бюджет (по данным налогоплательщика)».

Особенности заполнения некоторых листов

Титульный лист

Автоматически заполненные сведения на титульном листе отчета, необходимо проверить и, при необходимости, отредактировать. 

Раздел 2 «Расчет суммы налога по каждому транспортному средству (по данным налогоплательщика)».

Раздел 2 данной декларации может быть заполнен на основе авансового расчета по транспортному налогу за 9 месяцев. Для автоматического переноса данных из авансового расчета в годовую декларацию, следует нажать кнопку «Заполнить», которая находится на вкладке раздела 2. Программа автоматически найдет все авансовые расчеты за 9 мес. по выбранной ранее организации и указанному территориальному органа ФНС на титульном листе. В случае, если существуют более 1 экземпляра сохраненного расчета, то будет предложено выбрать расчет, из которого необходимо импортировать данные в годовую декларацию.

Внимание. При импорте данных из авансового расчета по транспортному налогу в годовую декларацию по транспортному налогу, колонка № 13 «Сумма льготы (руб.)» не будет перенесена в декларацию. В случае использования льгот, необходимо установить верные размеры льгот в годовой декларации.

Раздел 2 заполняется налогоплательщиком по всем транспортным средствам,  находящимся на территории соответствующего муниципального образования. При наличии у налогоплательщика транспортных средств на территории нескольких муниципальных образований, находящихся в ведении  налогового органа, в который представляется декларация,  Раздел 2 заполняется по каждому муниципальному образованию, на территории которого находятся  транспортные средства налогоплательщика.

Итоговое значение по расчету одного транспортного средства в колонке 14, автоматически переносится в раздел 1 в соответствующий лист с необходимым кодом ОКАТО, в случае, если в разделе 1 лист с нужным кодом ОКАТО отсутствует, то будет создан новый экземпляр раздела, с кодом ОКАТО указанным в разделе 2.

Раздел 1 «Сумма налога, подлежащая уплате в бюджет (по данным налогоплательщика)».

При наличии у налогоплательщика объектов налогообложения, находящихся на территории нескольких муниципальных образований, находящихся в ведении одного налогового органа, в котором осуществлена постановка на учет налогоплательщика по месту нахождения транспортных средств, заполняется один Расчет. При этом Раздел 1 заполняется по каждому муниципальному образованию, на территории которого находятся  транспортные средства налогоплательщика.

Данный раздел формируется автоматически, по данным раздела 2.
