Отчет "Журнал учета выданных счетов-фактур"

Продажа ® Ведение книги продаж ® Журнал учета выданных счетов-фактур

Отчет предназначен для формирования списка счетов-фактур и корректировочных счетов-фактур, выданных покупателям.

Отчет формируется в соответствии с правилами ведения журнала учета выданных счетов-фактур.

Для изменения параметров формирования отчета нужно снять флажок Сформировать отчет по стандартной форме.

  • Отображать счета-фактуры, не передаваемые покупателю - в реестр дополнительно включаются счета-фактуры, не передаваемые покупателю, например, по строительно-монтажным работам, выполненным хоз. способом, по суммовым разницам, при исполнении обязанностей налогового агента.
  • Отбирать по контрагенту - отражаются счета-фактуры, выданные покупателю или группе покупателей. Если покупатель не указан, то будут выводиться все выданные за период счета-фактуры.
  • Группировать по контрагентам - реестр счетов-фактур группируется по контрагентам.
