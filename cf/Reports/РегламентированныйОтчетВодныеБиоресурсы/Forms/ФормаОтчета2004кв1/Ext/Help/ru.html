Отчет «Сведения о полученной лицензии (разрешении) на пользование объектами водных биологических ресурсов, суммах сбора за пользование объектами водных биологических   ресурсов, подлежащих уплате в виде разового и регулярных взносов»

Настоящий отчет предназначен для подготовки сведений о полученных лицензиях (разрешениях) на пользование объектами водных биологических ресурсов, суммах сбора за пользование объектами водных биологических ресурсов, подлежащих уплате в виде разового и регулярных взносов, в соответствии с приказом МНС России от 15.06.2004 № САЭ-3-21/368@.

Порядок заполнения отчета

При заполнении отчета рекомендуется соблюдать такую последовательность подготовки листов:

1. Титульный лист.

2. Сведения о физическом лице (вторая страница титульного листа).

3. Раздел 2 «Сведения о суммах и сроках уплаты сбора по каждому объекту водных биологических ресурсов по полученной лицензии (разрешению)».

4. Раздел 1 «Сведения о подлежащих уплате в бюджет суммах сбора за пользование объектами водных биологических ресурсов по полученной лицензии (разрешению)».

Особенности заполнения некоторых листов отчета

Необходимо указать дату подписания отчета.

В зависимости от даты подписания отчета, при вводе кодов бюджетной классификации в тех разделах декларации, где это предусмотрено, для выбора будут выдаваться либо 20-значные КБК, если дата подписания равна или позже 1-го января 2005 ода, либо 7-значные КБК, если дата подписания - раньше 1-го января 2005 года.

Титульный лист

Следует указать сведения об организации и сведения о должностных лицах, которые не были заполнены автоматически. Автоматически заполненные сведения необходимо проверить и, при необходимости, отредактировать. 

Показатель «Основной государственный регистрационный номер (ОГРНИП) для индивидуального предпринимателя» заполняется в обязательном порядке с 1 января 2005 года.

Сведения о физическом лице (вторая страница титульного листа)

Заполняется только в отношении должностных лиц и уполномоченного представителя, у которых ИНН отсутствует или не указан на титульном листе.

Для автоматического заполнения сведений о должностных лицах по данным информационной базы и ФИО уполномоченного представителя по данным титульного листа следует открыть отчет. При этом, в случае отсутствия ИНН у должностного лица или у уполномоченного представителя, будут автоматически созданы дополнительные листы, содержащие данные по каждому из них.

Раздел 2 «Сведения о суммах и сроках уплаты сбора по каждому объекту водных биологических ресурсов по полученной лицензии (разрешению)»

Заполняется отдельно по каждому объекту водных биологических ресурсов по полученной лицензии (разрешению).  При наличии нескольких объектов водных биологических ресурсов необходимо заполнить раздел по каждому из них. Для создания дополнительных бланков по разделу следует воспользоваться группой управляющих элементов «Дополнительные страницы» в форме отчета. При нажатии кнопки «Добавить» добавляется новый бланк, при нажатии кнопки «Удалить» удаляется текущий бланк, а нажатие кнопок «<» и «>»  приводит к переключению между бланками.

По строкам 170 - 280 отражаются суммы регулярных взносов, подлежащие уплате равными долями по соответствующим срокам их уплаты в течение всего срока действия лицензии (разрешения). Суммы регулярных взносов и количество сроков их уплаты определяются автоматически в зависимости от установленных в строках 060 и 070 даты начала и даты окончания срока действия лицензии.

При заполнении строк 090 - 110 следует учесть, что значение показателя количества объектов водных биологических ресурсов отражается только по какой-либо одной из указанных строк в зависимости от ставки сбора, подлежащей применению при исчислении суммы сбора в отношении соответствующего количества объектов водных биологических ресурсов.

Раздел 1 «Сведения о подлежащих уплате в бюджет суммах сбора за пользование объектами водных биологических ресурсов по полученной лицензии (разрешению)»

Заполняется в отношении всех объектов водных биологических ресурсов, разрешенных к пользованию по полученной лицензии (разрешению).

Коды (код) бюджетной классификации, предлагаемые для выбора при заполнении строки «Код бюджетной классификации», зависят от даты подписания отчета, указанной в диалоге управления титульного листа декларации.

Если в качестве даты подписания декларации указана дата, включая или позже 1 января 2005 года, предполагается, что декларация представляется в 2005 году и налог будет уплачиваться также в 2005 году, поэтому при вводе КБК будут выданы для выбора 20-значные коды.

Если в качестве даты подписания указана дата, более ранняя, чем 1 января 2005 года, для выбора будут выданы «старые» 7-значные коды.

Значение показателя по строке 030 определяется как сумма значений по строкам 140 всех страниц раздела 2.

Значение показателя по строке 040 определяется как сумма значений по строкам 150 всех страниц раздела 2.

Значение показателя по строке 050 определяется как сумма значений по строкам 160 всех страниц раздела 2. 

Значение показателей по строкам 060 - 170 определяется как сумма значений по соответствующим строкам 170 - 280 всех страниц раздела 2.
