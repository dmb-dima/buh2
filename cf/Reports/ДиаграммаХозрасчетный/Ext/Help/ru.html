Отчет "Диаграмма"

Отчеты ® Диаграмма

Отчет предназначен для представления бухгалтерских данных по счетам в графическом виде. Диаграмма может быть построена по выбранному счету бухгалтерского учета.

Построение отчета выполняется по кнопке Сформировать.

Для сохранения настроек отчета для использования в дальнейшем предназначены кнопки сохранения и восстановления значений.

Параметры формирования отчета можно задать по кнопке Настройка.

  • На закладке Общее указываются основные параметры отчета: период, анализируемый счет, периодичность (по дням, неделям, декадам, месяцам и т.д.), вид данных (остатки или обороты по счету, дебет или кредит, сумма или количество).
  • На закладке Серии и точки задаются параметры источника данных для диаграммы:
    • Данные, которые образуют серии при построении диаграммы, например, валюта (евро, рубль и т.д).
    • Данные, которые образуют отдельные точки для серии диаграммы, например, дата.
    • Определяется режим ограничения максимального количества отображаемых серий диаграммы (поле Максимум серий). Данный параметр определяет количество серий, которое будет отображено в диаграмме.
  • На закладке Диаграмма задаются параметры отображения диаграммы: тип диаграммы, вид подписей для пояснения значения каждой точки диаграммы, параметры раздвижения секторов для круговых диаграмм. Также задается вывод окантовки, отображение заголовка, отображение легенды, отображение света (моделирование эффекта освещенности объемных диаграмм).
  • На закладке Отбор задается отбор данных диаграммы.
