Отчет «Форма №5»

Данная форма условно называется «стартовой», она открывается при вводе нового отчета.

Стартовая форма предназначена для:

  • выбора организации, по которой составляется отчет или для выбора группы организаций;
  • выбора периода формирования регламентированного отчета;
  • автоматического определения формы регламентированного отчета, соответствующей выбранному отчетному периоду.

«Стрелка вправо» / «Стрелка влево» - кнопки предназначены для выбора отчетного периода. Нажатие кнопок «прокручивает» периоды, за которым может составляться данный регламентированный отчет.

«Выбрать форму» - кнопка становится доступной, если за выбранный период регламентированный отчет может быть составлен по разным формам. В этом случае у пользователя появляется  возможность выбора редакции формы регламентированного отчета, не соответствующей указанному отчетному периоду. При выборе формы в нижней части диалога выдается краткое описание выбранной формы, например, реквизиты документа, которым она была утверждена.

«ОК» - открывает выбранную форму регламентированного отчета.

Выбор группы организаций

В бухгалтерской отчетности предусмотрен режим, в котором отчет может заполняться по данным нескольких организаций. Организации, по данным которых которых будет заполняться отчет, можно выбрать, используя управляющий элемент «Организация» в стартовой форме.

Для того, чтобы выбрать дополнительные организации, необходимо нажать кнопку выбора («») элемента управления «Организация» и в открывшейся форме выбора отметить флажками необходимые организации.

В этом случае, отчет будет суммировать показатели полученные из информационной базы по каждой выбранной организации, включая организацию, которая указана в стартовой форме. Список организаций будет сохраняться и восстанавливаться вместе с данными отчета. Изменить список организаций в уже открытом или восстановленном отчете нельзя. Для составления отчета по другому набору организаций необходимо ввести новый отчет, в котором указать требуемый набор организаций.
