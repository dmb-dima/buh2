Отчет "Регистр учета стоимости ТМЦ, списанных в отчетном периоде"

Отчеты ® Регистры налогового учета по налогу на прибыль ® Регистр учета стоимости ТМЦ, списанных в отчетном периоде

Регистр предназначен для обобщения информации о списании товарно-материальных ценностей (ТМЦ), в т.ч. списание ТМЦ без перехода права собственности и реализация товаров ранее отгруженных без перехода права собственности.

В отчете показывается общая стоимость списанных ценностей в разрезе единиц учета, направления использования ТМЦ и аналитика учета расходов по списанию ТМЦ.

Отчет формируется по данным корреспонденций по кредиту счетов учета ТМЦ (10, 41 и др.).

Каждая строка отчета соответствует одной корреспонденции счетов.

Параметры формирования отчета можно задать по кнопке Настройка. Например, отбор по виду ТМЦ или видам расходов, группировку по направлениям использования.

 

Примечание
Отчет нужно формировать только после выполнения регламентной операции закрытия месяца "Корректировка стоимости номенклатуры".
