Отчет «Акцизы (Приложение 5)»

Отчет «Акцизы (Приложение 5)» предназначен для подготовки декларации по акцизам на нефтепродукты, в соответствии с Приложением № 5 к приказу МНС России от 10 декабря  2003 г. № БГ-3-03/675@.

После вызова формы декларации из списка «Регламентированные отчеты», в диалоге управления отчетом следует, в первую очередь, указать организацию, для которой будет заполняться отчет.

Порядок заполнения

Рекомендуется соблюдать такую последовательность подготовки листов:

1. Титульный лист.

2. Сведения о физическом лице (вторая страница титульного листа).

3. Приложения №№ 1 - 5 к Декларации (в произвольной последовательности).

4. Раздел 2.1 «Расчет налоговой базы».

5. Раздел 2.2 «Расчет суммы акциза».

6. Раздел 2.3 «Расчет суммы акциза, подлежащей уплате по месту нахождения обособленного подразделения налогоплательщика».

7. Раздел 1 «Сумма акциза на нефтепродукты,  подлежащая уплате в бюджет, по данным налогоплательщика».

Особенности заполнения некоторых листов декларации

Титульный лист

Следует указать сведения об организации и сведения о должностных лицах, которые не были заполнены автоматически. Автоматически заполненные сведения необходимо проверить и, при необходимости, отредактировать.

Сведения о физическом лице (вторая страница титульного листа)

Заполняется только в отношении должностных лиц и уполномоченного представителя, у которых ИНН отсутствует или не указан на титульном листе.

Приложения  №№ 1, 2

Заполняются отдельно по каждому  поставщику.

В верхней части приложений указываются сведения о  поставщике.

При наличии нескольких поставщиков необходимо заполнить приложения отдельно по каждому из них. Для создания дополнительных бланков по приложению следует воспользоваться механизмом создания дополнительных листов, органы управления которого расположены в диалоге управления формированием отчета. При нажатии кнопки «Добавить» добавляется новый бланк, при нажатии кнопки «Удалить» удаляется текущий бланк, а нажатие на кнопки «<» и «>» приводит к переключению между бланками.

При наличии по одному виду нефтепродуктов нескольких счетов-фактур, по каждому счету-фактуре заполняется отдельная  строка.

Приложения  №№ 3, 4

Заполняются отдельно по каждому  покупателю.

В верхней части приложений указываются сведения о покупателе.

При наличии нескольких покупателей необходимо заполнить приложения отдельно по каждому из них. Для создания дополнительных бланков по приложению следует воспользоваться механизмом создания дополнительных листов, органы управления которого расположены в диалоге управления формированием отчета. При нажатии кнопки «Добавить» добавляется новый бланк, при нажатии кнопки «Удалить» удаляется текущий бланк, а нажатие на кнопки «<» и «>» приводит к переключению между бланками.

При наличии по одному виду нефтепродуктов нескольких счетов-фактур, по каждому счету-фактуре заполняется отдельная  строка.

Приложения  №5 «Сведения о суммах акциза, уплачиваемых по месту нахождения налогоплательщика и его обособленных подразделений»

Заполняется в случае уплаты налогоплательщиком сумм акциза по месту своего нахождения и по месту нахождения каждого своего обособленного подразделения.

В  отчете отдельно по каждому обособленному подразделению отражаются суммы акциза, уплаченные по  месту нахождения организации и по месту нахождения каждого обособленного подразделения.

Представляется в налоговый орган по месту нахождения организации. Копии представляются по  месту нахождения каждого обособленного подразделения.

Раздел 2.2  «Расчет суммы акциза»

В графу 5 по строкам 010 - 050 переносится налоговая база, рассчитанная в строках по коду 460 - 500 таблицы 2.1 «Расчет налоговой базы».

Раздел 2.3 «Расчет суммы акциза, подлежащей уплате по месту нахождения  обособленного подразделения налогоплательщика»

Объем продукции, реализованной на территории РФ по организации в целом переносится в графу 4 по строкам 030 - 070 из строк 610 - 650 таблицы 2.1 «Расчет налоговой базы».

Суммы акциза, подлежащие уплате в бюджет в целом по организации, переносятся в графу 6 по строкам 030 - 070 из строк 710 - 750 таблицы 2.2 «Расчет суммы акциза».

Сумма превышения налоговых вычетов над исчисленной суммой акциза переносятся в графу 6 по строкам 030 - 070 из строк 760 - 800 таблицы 2.2 «Расчет суммы акциза».

Раздел 1  «Сумма акциза на нефтепродукты,  подлежащая уплате в бюджет, по данным налогоплательщика»

В Разделе 1 отражаются суммы акциза, подлежащие уплате в бюджет, с разбивкой по кодам бюджетной классификации  (КБК),  на которые подлежат зачислению суммы акциза, рассчитанные в декларации.

По  каждому коду бюджетной  классификации  необходимо заполнять отдельно показатели по строкам, имеющим коды 010 - 030. Строки 030 заполняются на основании расчета, произведенного в пункте 23 таблицы N 2.2 раздела 2.

Если несколько сумм акциза,  рассчитанных по разным кодам вида подакцизного товара, подлежат зачислению на один код бюджетной классификации, следует сложить эти суммы и результат внести в строку по коду 030.

Коды (код) бюджетной классификации, предлагаемые для выбора при заполнении строки «Код бюджетной классификации», зависят от даты подписания отчета,
указанной в диалоге управления титульного листа декларации.

Если в качестве даты подписания декларации указана дата, включая или позже 1 января 2005 года, предполагается, что декларация представляется в 2005 году и налог будет уплачиваться также в 2005 году, поэтому при вводе КБК будут выданы для выбора 20-значные коды.

Если в качестве даты подписания указана дата, более ранняя, чем 1 января 2005 года, для выбора будут выданы «старые» 7-значные коды.
