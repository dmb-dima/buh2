Книга учета доходов и расходов предпринимателя

Отчеты ® Книга учета доходов и расходов предпринимателя

Отчет предназначен для формирования и вывода на печать Книги учета доходов и расходов и хозяйственных операций индивидуального предпринимателя по форме, утвержденной Минфином РФ.

Книга учета состоит из отдельных таблиц. Каждая таблица выводится на отдельной закладке. Каждая таблица может быть сформирована и напечатана отдельно от других. Для этого в открывшейся по кнопке Настройка форме нужно отметить те таблицы, которые нужно сформировать.

Если предприниматель осуществляет несколько видов деятельности и по каждому их них ведет отдельный том Книги учета, то в этом случае отчет следует формировать и печатать отдельно по каждому виду деятельности. Для этого в учетной политике на закладке Предприниматель нужно установить флажок Предприниматель ведет учет по нескольким видам деятельности.

Вид деятельности, для которого необходимо сформировать Книгу учета, указывается в поле Вид деятельности. При этом если деятельность предпринимателя не облагается НДС, то флажок Формировать по операциям, облагаемым НДС следует снять.

Для анализа операций только за последний месяц периода, за который формируется отчет, нужно установить флажок Выводить операции только за последний месяц периода.

Для расшифровки данных по документам поступления нужно установить флажок Выводить реквизиты документов поступления.
