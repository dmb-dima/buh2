Отчет «Налоговый расчет по авансовым платежам по транспортному налогу»

Настоящий отчет предназначен для подготовки расчета по авансовым платежам по транспортному налогу в соответствии с приказом Минфина РФ от 23.03.2006 № 48н (в редакции приказа Минфина РФ от 19.12.2006 г. № 179н).

Порядок заполнения

При заполнении расчета рекомендуется соблюдать такую последовательность подготовки листов:

1. Титульный лист.

2. Раздел 2 «Расчет суммы авансового платежа по транспортному налогу по каждому транспортному средству (по данным налогоплательщика)».

1. Раздел 1 «Сумма авансового платежа по транспортному налогу, подлежащая уплате в бюджет (по данным налогоплательщика)».

Особенности заполнения некоторых листов расчета

Титульный лист

Автоматически заполненные сведения на титульном листе отчета, необходимо проверить и, при необходимости, отредактировать. 

Раздел 2 «Расчет суммы авансового платежа по транспортному налогу по каждому транспортному средству (по данным налогоплательщика)»

Раздел 2 заполняется налогоплательщиком по всем транспортным средствам,  находящимся на территории соответствующего муниципального образования. При наличии у налогоплательщика транспортных средств на территории нескольких муниципальных образований, находящихся в ведении  налогового органа, в который представляется Расчет,  Раздел 2 заполняется по каждому муниципальному образованию, на территории которого находятся  транспортные средства налогоплательщика.

Итоговое значение по расчету одного транспортного средства в колонке 14 автоматически переносится в раздел 1 в соответствующий лист с необходимым кодом ОКАТО. В случае, если в разделе 1 лист с нужным кодом ОКАТО отсутствует, то будет создан новый экземпляр раздела с кодом ОКАТО, указанным в разделе 2.

Кнопка «Импортировать» предназначена для переноса данных в раздел 2 из годовой декларации по транспортному налогу.

Перенос возможен только если период формирования авансового расчета - 1 квартал. Для переноса данных в отчет за полугодие или 9 месяцев необходимо пользоваться функционалом создания отчета на основе существующего, путем копирования (клавиша F9) в журнале отчетов.

Раздел 1 «Сумма авансового платежа по транспортному налогу, подлежащая уплате в бюджет (по данным налогоплательщика)»

При наличии у налогоплательщика объектов налогообложения, находящихся на территории нескольких муниципальных образований, находящихся в ведении одного налогового органа, в котором осуществлена постановка на учет налогоплательщика по месту нахождения транспортных средств, заполняется один Расчет. При этом Раздел 1 заполняется по каждому муниципальному образованию, на территории которого находятся  транспортные средства налогоплательщика.

Данный раздел формируется автоматически по данным раздела 2.

Внимание!

Для конфигурации "Бухгалтерия предприятия 8", начиная с релиза 1.6.6, реализовано автоматическое заполнение отчета "Налоговый расчет по авансовым платежам по транспортному налогу" по данным информационной базы.

Для того, чтобы отчет заполнялся автоматически необходимо заполнить сведения о транспортных средствах, состоящих на учете организации. Данные заполняются в регистре сведений "Регистрация транспортных средств", кнопка вызова которого расположена на верхней панели меню управления отчетом. Автозаполнение отчета происходит при нажатии кнопки "Заполнить".

Сумма налоговой льготы автоматически не пересчитывается и если при автозаполнении отчета изменились данные участвующие в расчете суммы льготы, то её необходимо откорректировать вручную.
