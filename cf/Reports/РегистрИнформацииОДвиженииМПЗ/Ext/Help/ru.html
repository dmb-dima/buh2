Отчет "Регистр информации о движении МПЗ"

Отчеты ® Регистры налогового учета по налогу на прибыль ® Регистр информации о движении МПЗ

Регистр предназначен для обобщения информации о наличии и движении приобретенных материально-производственных запасов (МПЗ).

Отчет можно формировать по виду МПЗ. Предоставляется возможность выбрать режим формирования отчета:

  • по МПЗ на складах,
  • по МПЗ отгруженным,
  • по материалам в переработке.

Параметры формирования отчета можно задать по кнопке Настройка.
