Отчет «Декларация по водному налогу»

Отчет «Декларация по водному налогу» предназначен для подготовки декларации по водному налогу по форме, утвержденной приложением №1 к приказу Министерства финансов Российской Федерации от 03.03.2005 г. №29н.

После вызова формы декларации из списка «Регламентированные отчеты», в диалоге управления отчетом следует, в первую очередь, указать организацию, для которой будет заполняться отчет.

Порядок заполнения

При заполнении декларации рекомендуется соблюдать такую последовательность подготовки листов

1. Титульный лист.

2. Сведения о физическом лице (вторая страница титульного листа).

3. Раздел 2.1 «Расчет налоговой базы и суммы водного налога при заборе воды из водного объекта».

4. Раздел 2.2 «Расчет налоговой базы и суммы водного налога при использовании акватории водного объекта, за исключением лесосплава в плотах и кошелях».

5. Раздел 2.3 «Расчет налоговой базы и суммы водного налога при использовании водного объекта без забора воды для целей гидроэнергетики».

6. Раздел 2.4 «Расчет налоговой базы и суммы водного налога при использовании водного объекта для целей лесосплава в плотах и кошелях».

7. Раздел 1. «Сумма налога, подлежащая уплате по местонахождению объекта налогообложения (по данным налогоплательщика)» - заполняется автоматически.

Разделы 2, 2.1, 2.2, 2.3 и 2.4 представляется при наличии у организации соответствующего объекта налогообложения. 

Особенности заполнения некоторых листов декларации

Титульный лист

Следует указать сведения об организации и сведения о должностных лицах, которые не были заполнены автоматически при открытии отчета. Автоматически заполненные сведения необходимо проверить и, при необходимости, отредактировать.

Сведения о физическом лице (вторая страница титульного листа)

Заполняется только в отношении должностных лиц, у которых ИНН отсутствует или не указан на титульном листе.

Автоматическое заполнение сведений о должностных лицах по данным информационной базы происходит при открытии отчета или при изменении организации на форме декларации.

Раздел 1 «Сумма налога, подлежащая уплате по местонахождению объекта налогообложения (по данным налогоплательщика)»

Раздел 1 заполняется на основании данных разделов 2, 2.1, 2.2, 2.3 и 2.4. 

Для автоматического заполнения показателей раздела 1 необходимо предварительно заполнить разделы 2, 2.1, 2.2, 2.3,  2.4. Автоматическое заполнение производится по нажатию кнопки «Заполнить». 

Будьте внимательны, так как программа не будет проверять наличие текущих введенных сведений на данном листе декларации, а полностью заменит их данными из других разделов, если же такие данные не введены или введены не верно, то автоматического заполнения не произойдет.

Для каждого объекта административно-территориального деления, на территории которого налогоплательщик осуществляет пользование водным объектом создается отдельный лист раздела 1.

Местонахождение объекта водопользования по коду ОКАТО, указанному в разделах 2-2.4 декларации.

Автоматически заполняются   реквизиты:

  • Код бюджетной классификации (010);
  • Код ОКАТО (020);
  • Сумма налога, подлежащая уплате в бюджет (030).

При вводе КБК будут выданы для выбора 20-значные коды.

Раздел 1 заполняется налогоплательщиками в части сумм платы за пользование водными объектами, по месту нахождению объекта налогообложения (по данным налогоплательщика). 

При заполнении раздела 1 следует, при необходимости, скорректировать автоматически заполненные данные раздела, путем двойного нажатия мышки на требуемой области декларации.

Разделы 2-2.4

В разделах 2-2.4 по каждому водному объекту  (объекту налогообложения) заполняется отдельный лист. Вам должно быть уже известно, что в 1С многостраничные разделы могут содержать не ограниченное количество вставленных по необходимости страниц.
